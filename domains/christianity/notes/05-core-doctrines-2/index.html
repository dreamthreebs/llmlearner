    05 - 核心教义（下）— 救赎论、罪与恩典、末世论¶

    主维度：D2 核心教义
    关键关系：
    - 救赎论 (理论) --依赖--> 基督论 (理论)

    学习目标¶

    学完本章，你应能：

    1. 区分西方（奥古斯丁传统）与东方正教在原罪 / original sin上的典型表述，并说明为何这会影响对恩典与救恩的想象。
    2. 概述至少四种赎罪论 / theories of atonement的核心主张，并能指出各传统在历史上更常强调哪一种。
    3. 解释称义 / justification与成圣 / sanctification的差别，以及宗教改革时期天主教与新教争论的焦点。
    4. 描述基督教末世论 / eschatology的个人维度与宇宙维度，并简要比较千禧年主义 / millennialism三种读法。
    5. 在与佛教、道教的对话框架中，指出基督教救恩论与末世论在问题意识与时间结构上的差异（不作孰优孰劣判断）。

    可靠度标注说明（与本域 01-overview 一致）：Level 1 为学界广泛接受的事实性描述；Level 2 为主流学术观点；Level 3 为仍有争议的学术议题；Level 4 为特定教派或传统的自我理解，学术上常需与其他叙事并置。

    第一部分：罪（Sin）¶

    1.1 原罪（Original Sin）与人类的「堕落处境」¶

    Level 1：在基督教叙事中，始祖悖逆（传统上关联《创世记》中亚当、夏娃的故事）被理解为人类与上帝关系破裂的起源性事件——不仅是一次古代个案，更在神学上被读作整个人类处境的隐喻与根源。

    Level 2（西方拉丁传统，尤其奥古斯丁 Augustine）：奥古斯丁强调，堕落之后人类继承了一种败坏的性情 / fallen nature（「原罪」常被理解为这种状态 / condition，而非仅指个人所作的具体恶行）。在此框架下，人凭自身无法恢复与上帝的正当关系，救恩必须来自上帝的主动行动。

    Level 3（东方正教 Eastern Orthodox 的常见表述）：东方教父传统往往更强调人因堕落而承受必死性 / mortality与倾向罪 / tendency to sin，而对「继承罪责 / inherited guilt」的表述相对谨慎或与西方不完全同构。学术上常提醒：不宜用单一标签「东方教会都如此」概括所有希腊、斯拉夫、东方教会的讲法，但与西方刑事—法庭式隐喻的距离是反复出现的比较点。

    Level 4（教牧与礼仪层面）：各教会在婴儿洗礼、告解、补赎等实践上对「罪」的体认强度不同，但多数传统同意：罪不仅是「做错事清单」，而是与上帝隔绝并扭曲整全生命的力量。

    1.1a 补论：原罪如何「传递」？（西方思想史片段）¶

    Level 3：奥古斯丁曾借助当时生物学猜想讨论罪性经由世代传递（后世常误读为粗陋遗传学，应在古代科学框架内理解）。中世纪经院神学区分原罪与本罪 / actual sin（个人具体过犯），并讨论洗礼如何「除去原罪之罚」等精细命题。宗教改革时期，路德、加尔文一方面继承奥古斯丁遗产，一方面重新编排「律法—福音」与「因信称义」的讲法，使「罪」的诊断直接服务于唯独恩典的宣讲。

    Level 3（东方侧）：东方教父常强调亚当的悖逆带来死亡进入世界，而死亡是罪的主要后果；与上帝疏远的动态过程（及由此生的恶习）与西方「法庭式罪责账户」不必同构。当代正教神学家常批评西方将「法律隐喻」过度本体化，但西方神学家亦会回应：保罗书信中确有审判、称义等法庭语言——问题是如何与和好、联合、得胜等意象整合，而非只取一端。

    1.2 罪作为「与上帝隔绝」，而不只是道德失误¶

    Level 2：在系统神学语言中，sin 常被同时描述为：

    • 道德维度：违背律法或爱的命令；
    • 关系维度：背叛、疏远那位创造并呼召人的上帝；
    • 存在维度：一种向内转、以自我为中心的生存方式（传统词汇如 curvatus in se）。

    因此，基督教的「罪观」与单纯「社会规范违反」或「心理冲动」并不等同——尽管在现实分析中可以交叉。

    1.3 与佛教对比：无明与悖逆¶

    Level 3（比较宗教学框架）：佛教（尤其大乘常见讲法）常将根本问题归于无明 / avidyā——对缘起性空的不了知，由此起烦恼、造业、感苦。解脱路径核心是智慧与修行，强调因果与业（karma）的可塑性。

    基督教则强调人处在一种对造物主的错位关系中：不仅是「不知道」，更常包含意志上的抗拒或不忠（视传统而定，强调点有别）。用工程比喻：佛教像诊断「系统对真实数据结构理解错误导致连锁故障」；基督教像诊断「系统与设计者之间的信任与权限关系破裂，需由设计者发起修复协议」。

    此类对比是问题意识层面的，并不自动推出「哪一方更合理」——学术讨论关注的是内部一致性与历史演变。

    1.4 为何重要：问题的「严重度」塑造方案的「规模」¶

    Level 2：神学上常说：对罪与死的诊断越严厉，对十字架与复活的依赖就越强。反之，若将罪主要理解为可凭教育和制度渐进解决的社会问题，基督论与恩典的中心性在叙事上就会相对边缘——这在自由主义神学与现代政治神学中有历史实例。

    第二部分：救赎论（Soteriology）¶

    Level 1：Soteriology（希腊语 sōtēria，拯救）研究「基督之死与复活如何带来救恩」。这是基督教神学的核心议题之一，且高度依赖已在基督论中确立的命题：救主是谁、做了什么。

    2.1 若干经典赎罪理论（非排他列表）¶

    历史上没有任何一个大公会议把某一种理论单独定为全教会「官方公式」；不同世纪、不同传统侧重不同隐喻。下表便于入门对照（人物为历史上常与某说相关联的代表，细节学者仍有争论）：

    理论（英文常名） 核心意象 常与谁关联 学术备注
    赎价论 / Ransom theory 基督之死作为向撒但 / Satan 或某种奴役势力付出的赎价 / ransom 部分早期教父 Level 3：文本与哲学背景（希腊罗马解救叙事）解释纷纭
    满足论 / Satisfaction theory 冒犯上帝尊严与秩序，需补偿 / satisfaction；基督以顺服与献上生命满足父上帝的「荣誉 / honor」 安瑟伦 Anselm（11 世纪） Level 2：中世纪西欧封建—法庭隐喻极强
    刑罚替代论 / Penal substitution 罪人应受刑罚；基督代替 / in our place承担，使上帝的公义与怜悯得以同时彰显 常与宗教改革时期改革宗、部分福音派论述相连 Level 3：在伦理与隐喻上受现代哲学、伦理学广泛讨论
    道德影响论 / Moral influence 十字架首要显示上帝之爱，感动人悔改向善 阿伯拉尔 Abelard（12 世纪）等 Level 2：强调主观转化，常被与「客观成就」理论对照
    基督得胜论 / Christus Victor 基督之死与复活是对罪、死、魔鬼权势的胜利 / victory；释放被掳者 早期教会图像与礼仪；经 Aulén 等现代神学家再阐释 Level 2：在当代与政治解放神学等有对话空间

    Level 4 提醒：今日教派讲道中常出现「混合隐喻」——例如同时谈「满足上帝的公义」与「胜过黑暗权势」。学术描述应区分：历史上某一神学家的论证结构 vs 普通信徒的综合感受。

    2.1a 思想史线索：从教父到中世纪再到改革宗¶

    Level 2（粗线条叙事）：早期希腊语教父常在得胜、解救、医治、与神联合等意象中讲十字架；拉丁西方在中世纪盛期更发展出安瑟伦式满足论的精细逻辑；宗教改革在反对赎罪券与弥撒献祭经济时，常强化刑罚替代与唯独信心的配搭；启蒙以降，道德影响论与各类「非客观刑罚」读法在自由派神学中抬头；20 世纪 Aulén 等工作又推动学界重新审视古典得胜主题与政治隐喻（如压迫结构）之间的张力。

    Level 3：上述叙事是西欧中心的简化。拜占庭礼仪与斯拉夫传统的赞美诗学，从未以「安瑟伦是否成功」为轴心；将「东方=得胜、西方=法庭」二元对立，是教科书便利，不是现场实情。

    2.2 恩典（Grace）：非赚得的恩惠¶

    Level 1（圣经语句层面）：《以弗所书》2:8-9 常被引用：「你们得救是本乎恩，也因着信……不是出于行为，免得有人自夸。」（引文随译本略异）

    Level 2（神学定义）：Grace 指上帝不配得却赐予的善意与行动——救恩的起点与终极完成都被归于上帝主权，而非人的功德累积。

    Level 3（与佛教「业」的对比）：佛教内部宗派繁多，不能简化为一句「靠业力的宗教」。但在入门对照上：基督教主流传统强调人无法凭行为「赚取」与上帝的和好（尽管对「行为」在救恩中的位置有激烈争论，见下文）。佛教更常强调业与因果的可延续与可转变（依宗派不同，或重自力修行，或重他力救度，如净土传统）。比较时应注意：grace 与 karma 并不在同一语义平面，直接「对决」容易范畴错误。

    2.3 「唯独信心 / sola fide」还是「信心加以行为」？¶

    Level 2：

    • 新教主流（尤其路德宗、改革宗）：强调称义是上帝宣告性的礼物，通过信心领受；好行为是感恩与成圣的果实，而非称义的「工价」。
    • 罗马天主教：在特伦托会议（Council of Trent, 16 世纪）以来的正式神学表述中，反对将「信心」与「爱/行为」割裂；信心、望德、爱德在称义过程中有其位置——当代天主教与新教在若干文件（如 1999 年《关于称义教义的联合声明》）上已显示部分收敛，但教会法与礼仪实践层面仍存差异。
    • 东方正教：较少使用拉丁式的「称义—成圣」二分法庭语言，更强调神化 / theosis（与上帝联合、参与神圣生命）为整体目标。

    Level 3：任何「天主教只靠行为、新教只靠信心」的漫画式对立，都不符合严肃教义史。

    2.3a 《联合声明》（JDDJ）与 remaining differences¶

    Level 2：1999 年路德宗世界联盟与罗马教廷签署 Joint Declaration on the Doctrine of Justification（《关于称义教义的联合声明》），承认双方在因信靠恩典称义等表述上存在共同基础，同时以「共识—非解决差异—仍需讨论」的结构列出未决问题。

    Level 3：学术评论焦点包括：文件在多大程度上代表全球路德宗教会；「称义」一词是否在两种神学语法中真指同一结构；礼仪与圣事论、教阶权威等周边教义是否仍使「共识」呈分层状态。入门者只需记住：对话已发生，但制度与灵修实践层面的整合远未完成。

    第三部分：称义与成圣（Justification and Sanctification）¶

    3.1 称义（Justification）：法庭隐喻与礼物的争议¶

    Level 2：Justification 在保罗书信传统中常带法庭 / forensic色彩：上帝宣告信者「在祂面前为义」。关键争论在于：这义是「算为 / imputed」（归算给信徒）还是「注入 / infused」（灌入其存在与能力）？是瞬间事件还是与成长相连的过程？

    立场倾向 称义如何理解 常与谁关联
    归算义 / imputed righteousness 基督的义被「归给」罪人；上帝以此宣告为义 路德、加尔文传统；现代福音派常见表述
    注入义 / infused righteousness 圣灵注入爱德与恩典，使人真实变为义（并持续增长） 特伦托以来的天主教主流神学表述
    侧重联合 / union with Christ 强调与基督联合的本体性、参与性，弱化纯法庭式二分 部分当代新教与正教对话；东正教神化论

    Level 1（历史事实）：16 世纪宗教改革与天主教反改革之间的冲突，称义教义处于爆炸中心——不仅是抽象命题，更牵涉教宗权威、圣经诠释、圣礼功效与修会实践。

    3.1a 特伦托会议（Trent）与「Canons」文体¶

    Level 2：1545–1563 年的特伦托大公会议在称义、成圣礼、圣经正典等议题上形成驳斥性条文（canons），以「若有人说……，当受咒诅」的方式划定天主教正统边界。对改教家的若干命题（如称义仅因信、告解礼非上帝所立等）作出否定。Level 3：当代天主教与史学界的诠释强调：这些条文需放回当时辩论语境，不宜直接等同于今日天主教每一位神学家的精细立场。

    3.2 成圣（Sanctification）：成为圣洁的过程¶

    Level 2：各传统普遍承认：成圣是一生持续的进程——品格、意向、行为在圣灵工作下渐趋与上帝的旨意一致。争议较少集中在「是否成圣」，而在「成圣与称义的逻辑次序及与永生的关系」。

    Level 2：圣灵（Holy Spirit）的角色：传统上被理解为内住、引导、赐能力——使人能够结出「圣灵的果子」（如仁爱、喜乐、和平等，参见《加拉太书》5 章）。

    3.3 时间线：称义 → 成圣 → 荣耀化（Glorification）¶

    Level 2（教学常用模型）：

    1. 称义：常被描述为在时间中的「转折点」（归信洗礼等礼仪标记因人而异）；
    2. 成圣：漫长进程；
    3. 荣耀化 / glorification：末世论终点——复活身体、完全与上帝旨意一致（参见《罗马书》8:30 的链式表述在神学上的延伸用法）。

    此为教学脚手架，东正教与部分天主教灵修更强调整个过程的圣事性与奥秘性，不宜机械套用。

    3.3a 东正教关键词：神化（Theosis）¶

    Level 2：Theosis（又译神化、与上帝相似化）在东方灵修与神学中占据枢纽位置：救恩不仅是「罪得赦免」的消极面，更是参与神圣生命（常用《彼得后书》1:4「有分于神的性情」为经文支点）。与拉丁「注入义」有家族相似，但东方论述更少依赖法庭宣告的单一轴心，更多依赖圣礼、禁食、祈祷、图像所构成的整全「成圣生态」。

    Level 3：将 theosis 简单等同于泛神论，是常见误读；正统表述强调受造物永不丧失受造性，「神化」是恩典性的参与，非本体混同。

    第四部分：末世论（Eschatology）¶

    Level 1：Eschatology 字面上关于「末后之事」，同时涵盖个人命运（死后如何）与宇宙终局（历史如何收束）。

    4.1 个人末世论（Individual eschatology）¶

    主题 常见基督教表述 教派差异要点
    死亡 / death 灵魂与身体分离（用语与哲学背景各传统不一） 正教较少系统化为拉丁经院图式
    审判 / judgment 死后审判与/或末日审判（传统不同组合方式） Level 3：新约学者对「单一审判 vs 多重阶段」有讨论
    天堂 / heaven、地狱 / hell 与上帝同在的福乐 vs 隔绝与痛苦 隐喻字面程度、地狱是否可逆等，现代神学争议大
    炼狱 / purgatory 死后净化过程，为进入天堂预备 天主教传统教义；新教多数拒绝；东正教对类似观念态度复杂且术语不统一

    Level 4：民间宗教想象（火焰、空间式天堂地狱）常与教义文本及礼仪象征交织；学术分析需分层。

    4.1a 个别审判、普遍审判与「中间状态」¶

    Level 3：西方天主教与新教部分传统区分死后即刻的个别审判与末日普遍审判；东正教礼仪中亦充满「站立于可畏审判座前」的意象，但系统化方式不同。Level 3：关于灵魂睡眠 / soul sleep（死后无意识直至复活）与即时天堂/乐园的对立，在新教内部（尤其再洗礼派与部分福音派经卷解读）有长期争论，不宜概括为「全基督教立场」。

    Level 2（炼狱再述）：天主教教义将炼狱表述为蒙赦罪之人的净化过程，与「地狱永罚」区分；中世纪民间与文学艺术常放大其空间—时间想象。宗教改革领袖多批判炼狱与赎罪券、弥撒献祭经济的捆绑关系，从而否定教义本身或重新诠释相关经文（如《马加比二书》次经地位亦成争议点）。

    4.2 宇宙末世论（Cosmic eschatology）¶

    Level 2（多数传统共享的「故事弧线」语言）：

    • 基督再临 / Second Coming：历史并非无限开放，乃由基督亲自终结与更新；
    • 最后审判 / Final Judgment：公义与启示的终极场景；
    • 新天新地 / New Heaven and New Earth：受造界被更新而非简单废弃（参《启示录》21-22 章意象）；
    • 身体复活 / resurrection of the body：人整全性终得恢复——不仅是「灵魂不灭」的希腊化简化版。

    Level 3：宇宙论与自然科学对话（宇宙热寂、生态危机）推动当代生态末世论等议题——属 Level 4 前沿讨论，此处不展开。

    4.3 千禧年主义（Millennialism）：《启示录》第 20 章的三种读法¶

    Level 2：《启示录》20:1-6 中的「一千年」如何理解，形成三种典型类型学标签：

    类型 要点 基督再临与千年的关系（典型读法）
    前千禧年论 / Premillennialism 千年为将来地上的统治阶段 基督先再来，后开始千年
    后千禧年论 / Postmillennialism 福音改变世界，千年象征或对应一段「基督教化」时期 千年后基督再来（历史上与乐观时代精神相关，当代较少主流）
    无千禧年论 / Amillennialism 千年为象征，指当下教会时代或基督已胜过仇敌的「真实却非字面」时期 不重时间轴上的未来地上王朝

    Level 3：内部还有「历史前千」「时代论前千」等细分；学术上须避免把美国通俗「被提 / Rapture」叙事等同于全球基督教。

    4.3a 被提（Rapture）与通俗末世论（美国语境）¶

    Level 3：19 世纪以来，部分英美福音派将帖撒罗尼迦书信中的「被提到云里」与千禧年时间轴结合，形成灾前被提等复杂年表。此套叙事在流行文学与影视中影响极大，但Level 1：全球大多数基督教礼仪传统（天主教、东正教、主流新教宗派）并不以此套年表为教义核心。Level 4：学术上常将其置于现代性焦虑、冷战、中东政治的社会史框架中分析。

    4.4 与佛教末世论的结构性对照¶

    Level 3：佛教宇宙观常含轮回 / saṃsāra与极长时间尺度（劫），某些经典描述弥勒出世等，但整体上不同于基督教「创造—堕落—救赎—终末审判—新创造」的单向线性终局。佛教更多强调打破轮回或佛土等解脱论式终趣，而非单一宇宙性「最后法庭」。

    Level 3（再细化）：对个体而言，佛教常见目标是出离轮回或回入娑婆度生等多样典范；基督教则常以复活与永生在上帝国中为终局。对宇宙而言，佛教各派是否主张「法灭尽」、是否强调成住坏空等周期，与基督教一次性的终末论事件仍常呈类型差异——但大乘「净土」思想中他力救度与信愿因素，又成为比较神学家反复对照的接口（对照不等于等同）。

    4.5 与道教相关的对照视角¶

    Level 4（概括性较强，须谨慎）：道教传统中常见与道合同、长生、神仙境界、劫运更新等主题，侧重宇宙—身体—修炼的连续调理与阴阳气化的秩序。基督教末世论则强调上帝自外于受造秩序的行动（道成肉身、再临、复活）带来的断裂式更新。二者在「历史是否朝向单一终局事件」上常呈现不同预设——比较应回到具体经典与宗派，避免本质主义。

    Level 4（可操作的阅读提示）：道经中劫运 / kalpa-like cycles、太平治世、度人经教等母题，与基督教千禧年主义在社会心理层面有时被比较（期待终极转机），但本体论基础不同：一为道气与修炼技术所编织的宇宙论，一为创造主—受造物—救赎史所编织的叙事神学。工程背景读者可记：状态空间模型不同——变量定义、边界条件、收敛目标皆异。

    第五部分：天主教、东正教、新教在救赎论上的核心差异（简表）¶

    下表为教学简化（Level 2）；每一格内部仍有派别与当代神学讨论。

    议题 罗马天主教 Catholic 东方正教 Orthodox 新教 Protestant（泛指，内部差异大）
    权威来源 圣经 + 圣传；教宗不可错（在极限定条件下）等 圣经 + 圣传；大公会议；不承罗马教宗首位权 唯独圣经 / sola scriptura（解释各异）；无统一教宗
    称义 注入义、与圣事及爱德相连；反对「唯独信心」片面化（特伦托以来） 较少依赖法庭式称义语言；强调与上帝联合、神化 常强调唯独信心、归算义（路德宗、改革宗尤然）
    行为角色 信心与爱德合作；好行为在救恩秩序中有位置 善工为悔改与联合的果实，非「赚功德」口语化理解可争议 行为为成圣果实、称义确据的讨论复杂；反「靠行为称义」是常见修辞
    圣礼与救恩 七件圣事；洗礼、告解、圣体等与赦罪、恩典密切关联 奥秘（圣礼）为教会生命的核心；洗礼、圣餐、傅油等 通常只认洗礼、圣餐为圣礼；对「圣礼性」与「纪念性」强调不一
    圣母与圣徒 圣母无原罪（教义）、转求圣徒等 尊敬圣母、圣徒相通，神学表述与拉丁传统有别 多数派拒绝向圣徒祈祷；基督为唯一中保的强调度高
    末世论 炼狱、特定末世论教义文件较系统 强调奥秘、复活与审判，细节常留待礼仪与图像 千禧年解读多样；一般拒绝炼狱

    议题 简述
    地狱观 永刑论、有条件不朽、普救论等争论，牵涉圣经诠释与上帝公义概念
