# 点集拓扑（Point-Set Topology）

> 创建日期：2026-03-07

## 背景与起点

- **已有知识**：本科数学分析基础（极限、连续、$\epsilon$-$\delta$），集合论基础（交并补、映射），但不深
- **从哪开始**：拓扑空间的定义
- **目的**：纯兴趣，重点在核心定理及其思想

## 领域概览

点集拓扑研究的是：**什么性质只依赖于"哪些点彼此靠近"这件事，而和具体的距离无关？**

在数学分析中，连续性、收敛性、紧致性这些概念都依赖于度量（距离函数）。但拓扑学的核心洞察是：这些概念其实不需要距离——只需要一个更基本的结构：**开集族**。拓扑空间就是"只保留了开集结构，丢掉了距离"的空间。

点集拓扑是所有现代数学中涉及"连续性"概念的基础——微分几何、代数拓扑、泛函分析、代数几何都建立在它之上。

## 学习路径

1. **拓扑空间与开集** — 整个理论的起点
2. **连续映射与同胚** — 拓扑学关心什么样的变换
3. **紧致性** — 最重要的性质之一，Heine-Borel 的推广
4. **连通性** — 空间是否"一整块"
5. **分离公理** — 空间"有多好"的层级体系
6. **乘积空间与 Tychonoff 定理** — 最深刻的定理之一
7. **度量化定理** — 什么时候拓扑空间其实是度量空间
