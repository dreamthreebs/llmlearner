    04 - 句法学（Syntax）¶

    主维度：D1 语言分析层次
    次维度：D2 理论视角
    关键关系：
    - 句法学 (理论) --依赖--> 形态学 (概念)
    - 形式主义 (理论) --用于--> 句法学 (理论)
    - 语义学 (概念) --依赖--> 句法学 (理论)

    学习目标¶

    读完本章，你应该能够：

    1. 用自己的话说出句法学研究什么，以及它与语序、可接受性之间的关系。
    2. 用至少两种成分测试论证某一串词是否构成句法成分，并画出简单的短语结构树。
    3. 解释中心词、X-bar 模板与中心词参数（head-initial / head-final）在跨语言比较中的作用。
    4. 读出 Greenberg 式语序类型（SVO、SOV 等），并结合 WALS 数据理解“常见≠唯一”。
    5. 区分形式主义与功能主义在“语法从何而来”这一问题上的基本立场（不要求掌握技术细节）。

    Part 1：什么是句法学？¶

    1.1 定义：词以上的组合规律¶

    句法学（syntax）研究的是：词（words）如何按规则组合成更大的单位——如短语（phrase）、小句（clause）、句子（sentence），以及哪些组合在母语者看来是“合法的”。

    可以把它想成一种结构工程：你已经有了“砖块”（词，形态学关心砖块怎么造），句法学关心“哪些搭法稳、哪些会塌”。

    可靠程度：Level 1（教科书共识）

    1.2 句法规则 vs 仅仅是语序¶

    日常说法里，“语法”常被等同于“语序”。在语言学里，语序（word order）只是句法的一个侧面：真正关键的是隐藏的组合规则与层次结构（后面 Part 2 展开）。

    但语序仍是重要线索：同一组词，顺序不同，可接受性往往不同。

    1.3 语法性判断（grammaticality judgment）¶

    语法性（grammaticality）指：在理想化的语言能力层面，一句话是否符合母语者内化的句法规则。

    • 合语法（grammatical）的例子，常不加标记。
    • 不合语法的例子，语言学惯例在句首加星号 * 表示不可接受（unacceptable）。
    例句 说明
    The cat sat on the mat. 英语母语者通常判断为自然、合法
    *Cat the on mat sat the 词相同但组合非法；标记为 *

    注意：可接受性（acceptability）还会受记忆、语体、修辞影响；课堂上常把“语法性”当作理论上的干净概念，把“听起来怪但能理解”与“彻底不合法”区分开。

    可靠程度：Level 1–2（Level 2：真实实验里星号判断要配合统计与语境控制）

    1.4 “知道语法”≠“能说出规则”：内隐知识（tacit knowledge）¶

    母语者大多能直觉判断一句话是否别扭，却常常无法准确陈述规则。

    语言学把这种现象与语言能力（competence）联系起来：规则可能以内隐（tacit / implicit）形式存储在大脑中，就像你会骑自行车却不一定能写出力学方程。

    句法学的一部分工作，就是把这种内隐规律外显化为可检验的假说（如短语结构规则、树形图、约束）。

    Part 2：成分结构（Constituency）¶

    2.1 线性顺序只是表面：句子有层次¶

    词在纸上是从左到右线性排列的，但句法分析的核心主张是：词与词之间组成嵌套的成分（constituents）。

    成分（constituent）指：在句法结构中作为一个整体参与规则的单位（可移动、可替换、可省略等）。

    2.2 成分测试（constituency tests）¶

    下列测试在教学中常用；没有单一测试是绝对的，通常要多种证据一致。

    （1）替代测试（substitution）¶

    若能用一个形式简单的代词（或其它占位成分）替换一串词，且句子仍合法，这串词很可能是一个成分。

    英语：

    • The tall boy arrived.
    • He arrived.（用 he 替换 the tall boy）

    → 有理由认为 the tall boy 是一个名词短语成分（NP）。

    中文也可用类似思路（代词、指示词、省略后是否自然）：

    • 那只大黄猫睡在沙发上。
    • 它睡在沙发上。

    → 那只大黄猫 很可能是一个 NP 成分。

    （2）移动测试（movement / cleft 等）¶

    把一串词整体移到强调结构等特殊位置后仍合法，常说明它是一个成分。

    • It was [the tall boy] that came.

    （3）省略测试（ellipsis）¶

    在并列等结构中，若某部分可省略且剩余部分平行合法，被省略的常对应一个成分。

    • Mary read a book, and John did too.（did too 省略了 read a book 一类 VP）

    可靠程度：Level 2（教学常用；真实语言现象有例外与方言差异）

    2.3 短语结构规则（Phrase Structure Rules, PSR）¶

    短语结构规则用改写规则（rewrite rules）说明：一个符号如何展开成若干符号。

    下面是极度简化的英语教学模型（真实分析会更细，且因理论而异）：

    规则 读法（直觉）
    S → NP VP 句子 S 由一个 NP 和一个 VP 组成
    NP → (Det) (AdjP) N 名词短语可有冠词、形容词、名词（括号表示可选）
    VP → V (NP) (PP) 动词短语可有宾语 NP、介词短语 PP 等
    PP → P NP 介词短语 = 介词 + NP

    Det = determiner（限定词），如 the, a；AdjP = 形容词短语；P = preposition（介词）。

    2.4 树形图（Tree Diagrams）¶

    树形图用节点与分叉表示支配关系（谁组合成谁），而不是只画词序。

    英语例句：The cat chased the mouse.

    下面用缩进文本树示意（标签常见写法；不同教材细节可能不同）： 

            S
       / \
     NP   VP
    /|\    \
 Det N   V   NP
  |  |   |  /|\
 the cat chased Det N
              |  |
             the mouse

    读图约定（详见下文公式速查卡）：线段连接表示直接组成关系；同一母节点下的子节点通常按语序从左到右排列。

    2.5 中文树与英文的异同（示意）¶

    中文没有英语的 the/a 一类冠词，Det 常为空；定语多在名词前（如形容词、指示词、数量词）。

    例：那只大黄猫（简化为 NP） 

              NP
        /  |  \
     Det  AdjP  N
      |    |    |
     那只  大黄   猫

    与英语 NP 的相似点是：都以名词 N 为核心，周围可加修饰成分；差异主要在哪些功能词必须出现、修饰语相对中心语的位置。

    可靠程度：Level 2（具体树形依赖学派与分析颗粒度）

    Part 3：词组与中心词（Phrases and Heads）¶

    3.1 中心词（head）¶

    在短语 XP（X phrase）里，中心词（head）是决定该短语核心语法范畴的那个词：

    • NP 的中心词是 N（名词）
    • VP 的中心词是 V（动词）
    • PP 的中心词是 P（介词）
    • AdjP 的中心词是 Adj

    直觉：拿掉中心词，短语的“名词性/动词性”往往就站不住。

    3.2 X-bar 理论（X-bar Theory）——极简版¶

    X-bar 理论试图给所有短语一个统一骨架，而不是每种短语一套特殊画法。

    常见教学模板（X 代表 N/V/A/P 等范畴）：

    模板 含义（直觉）
    XP → Spec X' 短语 XP 可有标志语（Specifier, Spec）与中间层 X'
    X' → X Complement 中心词 X 先与补语（Complement）组合成 X'
    X' → X' Adjunct 可有附加语（Adjunct）；入门常先略去

    补语（complement）是与中心词选择关系最强的成分（如动词要求的宾语）；标志语位置常容纳主语、限定词短语等——具体放哪因理论版本而异。

    可靠程度：Level 2（教学简化；最简方案 Minimalism 中结构又有调整）

    3.3 中心词参数（Head Parameter）¶

    中心词参数描述：中心词相对于补语，是出现在补语前还是后。

    类型 含义 VP 例子（动词与宾语）
    Head-initial 中心在前，补语在后 英语：eat [fish] — V + 宾语
    Head-final 中心在后，补语在前 日语：[魚を] 食べる — 宾语（带格助词）+ V

    中文常被描述为混合：

    • VP 里动词常在后接宾语，整体像 head-initial：吃鱼、看书。
    • NP 里修饰语多在名词前：红色的花、三本书 —— 修饰语相对名词更像“中心在后、修饰在前”（与英语 red flowers 表面相似，但功能范畴分配仍要细分析）。

    因此：不要用单一语序标签代替整棵树的 head 方向；类型学上常分项描写（动词—宾语、名词—形容词、介词—名词等）。

    可靠程度：Level 2–3（Level 3：具体语言分析存在争议）

    Part 4：语序类型学（Word Order Typology）¶

    4.1 Greenberg 的基本语序类型¶

    在陈述小句里，若区分主语 S（Subject）、宾语 O（Object）、谓语动词 V（Verb），理论上 6 种基本排列：

    SOV, SVO, VSO, VOS, OVS, OSV

    Joseph Greenberg 的经典工作指出：并非所有组合都同样常见，且语序与其它性质（介词前后置、从句位置等）常有统计关联（蕴涵共性思路）。

    可靠程度：Level 2（教科书共识）；具体蕴涵关系的强度随样本与统计方法变化 → Level 3–4 前沿讨论见类型学专著

    4.2 世界语言的分布：WALS 81A 计数与比例¶

    WALS（World Atlas of Language Structures）的 Feature 81A（Dryer）给出各语言在样本中的归类计数（访问日期以数据库为准）。下列数字来自该特征页的公开计数（总和 1376）：

    类型 语言数 占全部样本比例（约）
    SOV 564 41.0%
    SVO 488 35.5%
    VSO 95 6.9%
    VOS 25 1.8%
    OVS 11 0.8%
    OSV 4 0.3%
    无单一优势语序 189 13.7%

    读数提醒（重要）：

    1. 这是样本不是全球人口加权；语系多样性会影响“语言数”统计。
    2. “无单一优势语序”说明：语序类型标签不能硬套每一种语言。

    来源：WALS Feature 81A。

    可靠程度：Level 2（数据库方法透明，但统计解释需谨慎）

    4.3 语序和谐（word order harmony）¶

    语序和谐指：某些结构倾向“同向”——例如 SOV 语言更常使用后置词（postposition，类似“学校里”这种后置功能词）而非前置介词；从句相对主句的位置也常呈现可统计的倾向。

    这不是说“没有反例”，而是类型学上的倾向，用来支持“结构之间存在系统关联”的假说。

    4.4 中文：SVO 与“像 SOV 的面貌”¶

    现代汉语常归类为 SVO（如：我吃饭）。但 把字句 等现象会让宾语出现在动词前，例如：

    • 我 把书 放在 桌上。

    粗线条分析：把书 把宾语话题化/受影响对象化，动词短语后部仍可带处所等补语；这说明：表层语序会受结构句式、信息结构、体貌等因素塑造，不能只用三个字母概括所有句子。

    Part 5：形式主义 vs 功能主义视角¶

    5.1 生成语法与普遍语法（Generative Grammar / UG）¶

    生成语法（Generative Grammar；常与 Chomsky 传统联系）倾向于认为：

    • 语法是一个形式规则系统，具有一定自治性（syntax is not just a mirror of meaning or use）。
    • 儿童在输入有限的情况下仍能习得复杂规则，暗示存在先天语言机能与普遍语法（Universal Grammar, UG）的假设空间。

    句法学在这一传统里高度形式化：树、移动、约束、特征一致等。

    可靠程度：Level 2（作为学派框架介绍）；UG 的具体内容与证据强度在学界有 Level 3–4 争议

    5.2 功能主义（Functionalism）¶

    功能主义更强调：交际功能、处理难度、频率、象似性等塑造形式。

    • 语法规则被视为在使用中被磨合成今天的样子，而不是完全独立的“纯数学系统”。
    • 更常问：为什么这种结构在沟通或认知上“好用”？

    5.3 简要对比（入门级别）¶

    维度 形式主义（粗线条） 功能主义（粗线条）
    核心隐喻 规则系统、模块 交际压力、使用中的优化
    典型问题 哪些句子结构合法？需要哪些抽象机制？ 为什么这种结构更常见、更易学？
    与跨语言共性 UG 参数、原则 类型学倾向、处理与语义动因

    本章不要求你在两派之间“选边”，但应能识别：同一语序事实，可以接不同的解释议程。

    公式速查卡¶

    记号 / 模板 含义（中文）
    S → NP VP 句子展开为主语名词短语 + 动词短语
    NP → (Det) (AdjP) N 名词短语：可选限定词、形容词短语 + 名词
    VP → V (NP) (PP) 动词短语：动词 + 可选宾语、介词短语
    PP → P NP 介词短语：介词 + 名词短语
    * 句首星号：不合语法 / 不可接受的例子
    XP → Spec X' X-bar：短语 XP 含标志语与 X' 层
    X' → X Complement X' 由中心 X 与补语构成
    Head-initial 中心词在补语前（如英语 V–Object）
    Head-final 中心词在补语后（如日语 Object–V）
    树（母—子） 母节点直接支配子节点；子节点从左到右常对应语序
