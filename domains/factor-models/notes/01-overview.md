## 1. 多因子模型在干什么

### 一句话版本

多因子模型试图回答：**为什么有些股票的长期收益率比另一些高？**

### 稍长版本

假设你有两只股票：

- 股票 A：过去 30 年年均收益 15%
- 股票 B：过去 30 年年均收益 8%

为什么差这么多？可能的答案：

1. **纯运气**——A 碰巧赶上好时候了
2. **风险补偿**——A 的风险更高，所以投资者要求更高的收益作为补偿
3. **市场无效**——市场犯了错，A 被低估了

因子模型的核心思路是**答案 2**：收益率的差异是**风险的补偿**。不同因子代表不同类型的风险。

> 但答案 3（行为金融学派的立场）也有很多支持者——这是这个领域最大的学术争论之一。

## 2. 从日常直觉到数学模型

### 2.1 什么是"因子"

**因子（factor）**是一种能**系统性地解释**大量资产收益率差异的变量。

直觉类比：你想预测一个人的收入。单变量（如学历）能解释一部分，但加上工作年限、行业、地区等多个变量能解释更多。每个变量就是一个"因子"。

在金融中：
- 因子是影响**一大批**股票收益率的共同驱动力
- 不是只影响一只股票的个别原因（那是"特异性风险"）

### 2.2 核心公式预览

多因子模型的一般形式：

$$r_i - r_f = \alpha_i + \beta_{i1} f_1 + \beta_{i2} f_2 + \cdots + \beta_{iK} f_K + \epsilon_i$$

| 符号 | 含义 |
|------|------|
| $r_i$ | 股票 $i$ 的收益率 |
| $r_f$ | 无风险利率（risk-free rate，如国债利率——投资者不承担任何风险就能获得的收益率） |
| $r_i - r_f$ | 超额收益（excess return）——超出无风险利率的部分，才是需要解释的 |
| $\alpha_i$ | 截距项——如果模型完美，$\alpha$ 应该为 0。$\alpha \neq 0$ 意味着模型没有完全解释收益 |
| $\beta_{ik}$ | 股票 $i$ 对因子 $k$ 的**暴露**（exposure）或**敏感度**——$\beta$ 越大，这个因子对该股票的影响越大 |
| $f_k$ | 第 $k$ 个因子的收益率（因子溢价，factor premium） |
| $\epsilon_i$ | 残差——模型无法解释的部分（特异性风险） |

> **你已经会的**：这就是多元线性回归（参见 `domains/cosmological-statistics/notes/03-likelihood-mle.md`），只是变量换成了金融概念。$\beta$ 就是回归系数，$\epsilon$ 就是残差。

### 2.3 最简单的例子：CAPM

CAPM（Capital Asset Pricing Model，资本资产定价模型）是最简单的因子模型——只有**一个因子**：

$$r_i - r_f = \beta_i (r_m - r_f) + \epsilon_i$$

- $r_m$ = 市场组合（market portfolio，可以简单理解为"所有股票的加权平均"，实操中通常用沪深300或标普500指数代替）的收益率
- $r_m - r_f$ = 市场超额收益——**唯一的因子**
- $\beta_i$ = 股票 $i$ 的市场 beta——衡量这只股票的收益率跟市场涨跌的同步程度

CAPM 说：一只股票的预期收益率**只取决于它的市场 beta**。beta 越高（跟市场越同步），收益率越高。

**问题**：实证检验发现 CAPM 不太对——很多高收益的股票组合并不是高 beta 的。需要更多因子。这就是 Fama-French 三因子模型和后续发展的动机。

## 3. 时间线速览

| 时期 | 年代 | 关键事件 |
|------|------|---------|
| 投资组合理论 | 1952 | Markowitz 提出均值-方差优化框架——"不要把鸡蛋放在同一个篮子里"的数学化 |
| CAPM | 1964-66 | Sharpe、Lintner、Mossin 独立提出 CAPM |
| CAPM 的实证困境 | 1970s-80s | 大量"异象"（anomaly）被发现——CAPM 无法解释的收益规律 |
| APT | 1976 | Ross 提出套利定价理论（APT）——允许多个因子 |
| 三因子模型 | 1993 | Fama & French 提出市场+规模+价值三因子模型 |
| 四因子 | 1997 | Carhart 加入动量因子 |
| 五因子模型 | 2015 | Fama & French 加入盈利能力和投资模式因子 |
| 因子动物园 | 2010s | 学术界发现了数百个"因子"——大部分可能是假的（p-hacking） |
| 机器学习因子 | 2020s | 用 ML 方法发现和组合因子 |

> **可靠度**：Level 1。时间线是学术共识。

## 5. 核心概念预览

### 风险溢价（Risk Premium）

投资者因为承担了某种风险而获得的**额外收益**。

$$\text{风险溢价} = \text{预期收益率} - \text{无风险利率}$$

无风险利率是"什么风险都不承担"的收益率（通常用短期国债利率代表）。任何超出这个的收益，都是在为某种风险"付费"。

### 异象（Anomaly）

CAPM 预测所有股票的预期收益率只取决于 beta。但如果有一组股票的收益率**系统性地**高于 CAPM 的预测，那就是一个"异象"——CAPM 解释不了的规律。

经典异象：
- **规模效应**：小市值公司的收益率系统性高于大市值公司
- **价值效应**：低市盈率（便宜的）股票收益率系统性高于高市盈率（贵的）股票
- **动量效应**：过去涨得好的股票继续涨，跌得多的继续跌

这些异象后来被整合为"因子"——Fama-French 的三因子和五因子模型就是把异象变成了系统性的风险解释。

### Alpha 和 Beta

| 概念 | 含义 | 投资者想要什么 |
|------|------|---------------|
| **Beta（$\beta$）** | 股票对某个因子的暴露程度 | 可以通过因子组合"买到"——便宜 |
| **Alpha（$\alpha$）** | 因子模型无法解释的超额收益 | 这才是真正的"本事"——非常稀缺 |

整个量化投资行业可以理解为：**区分哪些收益是 beta（因子暴露的补偿，任何人都可以通过被动策略获得），哪些是 alpha（真正的投资技能）**。

### 公式速查卡

| 公式 | 含义 |
|------|------|
| $r_i - r_f = \beta_i (r_m - r_f) + \epsilon_i$ | CAPM：收益只由市场 beta 决定 |
| $r_i - r_f = \alpha_i + \sum_k \beta_{ik} f_k + \epsilon_i$ | 多因子模型：收益由多个因子决定 |
| $\text{Sharpe Ratio} = \frac{E[r] - r_f}{\sigma}$ | 夏普比率：每单位风险获得的超额收益（$\sigma$ 是收益率的标准差） |
