# CMB（宇宙微波背景辐射）

> 创建日期：2026-03-07

## 背景与起点

- **已有知识**：本科物理（力学、电磁、热统、量子力学），无宇宙学基础
- **从哪开始**：膨胀宇宙基础

## 领域概览

CMB 是宇宙大爆炸约 38 万年后释放出的热辐射"余晖"。当时宇宙冷却到 ~3000K，电子与质子复合为氢原子，光子不再被频繁散射，宇宙变得透明。这些光子自由传播至今，被红移到微波波段（~2.725K）。

CMB 是一张宇宙婴儿期的"照片"。它几乎是完美的黑体辐射，但存在约 $10^{-5}$ 量级的温度涨落。这些微小涨落编码了早期宇宙的物理信息——物质密度、组成成分、几何结构，是现代精密宇宙学的基石。

### Planck 卫星观测的 CMB 全天温度图

![CMB 全天温度图](images/planck_cmb_2018.jpg)

红色 = 比平均温度稍热，蓝色 = 稍冷。温差仅约 ±200 μK（平均 2.725K 的十万分之一）。这些涨落是今天所有宇宙大尺度结构的种子。

## 学习路径

1. **膨胀宇宙** — 没有这个框架，后面都没法谈。核心是 Friedmann 方程。
2. **热历史** — 理解宇宙是从极热极密演化来的，各种粒子何时退耦。
3. **复合与退耦** — 回答"CMB 为什么存在"这个根本问题。
4. **扰动与声波振荡** — CMB 物理的核心机制。重子-光子流体中的驻波。
5. **角功率谱** — 把物理过程和观测量连接起来。理解峰的位置和高度为什么能测宇宙学参数。

## 核心机制

### 1. 膨胀宇宙

广义相对论 + 宇宙学原理（均匀各向同性） → FRW 度规：

$$ds^2 = -dt^2 + a(t)^2 \left[\frac{dr^2}{1-kr^2} + r^2 d\Omega^2\right]$$

其中 $a(t)$ 是尺度因子，描述空间整体的膨胀。Friedmann 方程把 $a(t)$ 的演化和物质-能量内容联系起来：

$$H^2 \equiv \left(\frac{\dot{a}}{a}\right)^2 = \frac{8\pi G}{3}\rho - \frac{k}{a^2}$$

**心智模型**：把宇宙想象成一块均匀膨胀的面团，$a(t)$ 描述面团的"缩放因子"，$H$ 是膨胀速率。不是物质在空间中运动，而是空间本身在膨胀。

### 2. 复合与退耦

宇宙早期温度极高，物质是电离等离子体（电子+质子+光子耦合在一起）。光子被自由电子频繁 Thomson 散射，走不远——宇宙是"不透明"的。

当温度降到 ~3000K（$z \approx 1100$），电子被质子捕获形成氢原子（复合），自由电子急剧减少，光子的平均自由程暴增到超过宇宙尺度——宇宙变透明。

这些"最后一次被散射"的光子就是我们今天观测到的 CMB。

### 3. 声波振荡（核心）

退耦之前，重子（质子+电子）和光子通过 Thomson 散射紧密耦合，形成一种"重子-光子流体"。这个流体中有两个竞争力：

- **辐射压**（光子提供）→ 阻止坍缩
- **引力**（暗物质提供的势阱）→ 驱动坍缩

结果：流体在暗物质势阱中做声波振荡——被引力拉入势阱，被辐射压弹回，如此反复。

退耦那一刻，振荡被"冻结"。正好处于最大压缩的那些模式、正好处于最大稀疏的模式，在 CMB 温度图上留下了特征印记。

### 4. 角功率谱 $C_\ell$

CMB 温度涨落 $\Delta T(\hat{n})/T$ 在天球上展开为球谐函数：

$$\frac{\Delta T}{T}(\hat{n}) = \sum_{\ell m} a_{\ell m} Y_{\ell m}(\hat{n})$$

$C_\ell = \langle |a_{\ell m}|^2 \rangle$ 描述角度尺度 $\theta \sim 180°/\ell$ 上涨落的功率。

$C_\ell$ 对 $\ell$ 的图上可以看到一系列峰：
- **第一峰**（$\ell \approx 220$）：退耦时恰好完成半个振荡周期的模式。其位置取决于宇宙几何（平坦 vs 弯曲）。
- **奇数峰 vs 偶数峰的高度比**：编码了重子密度 $\Omega_b$。
- **整体峰的包络衰减**：Silk 阻尼，光子扩散抹平了小尺度涨落。
