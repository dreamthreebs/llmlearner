    策略理论（上）— 动量、均值回复、趋势跟踪¶

    主维度：D3 策略理论 次维度：D2 数学工具（OU 过程），D4 风险与评估（策略评估指标） 关键关系： - 动量策略 (方法) --对比--> 均值回复策略 (方法) - 趋势跟踪 (方法) --是一种--> 动量策略 (方法) - OU 过程 (理论) --用于--> 均值回复策略 (任务)

    前置知识： - 01-overview（策略分类概览、Sharpe 比率、MDD） - 02-markets-and-data（OHLCV、收益率计算） - 03-math-tools-1（随机游走、平稳性、AR 模型——\(\phi_1\) 的正/负自相关含义） - 04-math-tools-2（协整——统计套利的基础、GBM）

    1. 策略的数学框架¶

    1.1 信号 → 仓位 → 收益¶

    所有量化策略都可以用一个统一的数学框架描述：

    \[r_{\text{strat},t} = \sum_{i=1}^{N} w_{i,t} \cdot r_{i,t}\]

    符号解释： - \(r_{\text{strat},t}\)：策略在第 \(t\) 天的收益率 - \(N\)：交易的资产数量 - \(w_{i,t}\)：第 \(t\) 天在第 \(i\) 个资产上的权重（仓位比例） - \(r_{i,t}\)：第 \(i\) 个资产在第 \(t\) 天的收益率

    关键：策略之间的区别不在于收益率 \(r_{i,t}\)（这是市场给的），而在于权重 \(w_{i,t}\)（这是你的决策）。

    权重的含义： - \(w_i > 0\)：做多——持有该资产，价格涨了赚钱 - \(w_i < 0\)：做空——借入并卖出该资产，价格跌了赚钱 - \(w_i = 0\)：不持有该资产 - \(\sum |w_i| = 1\)（归一化）或 \(\sum w_i = 0\)（多空对冲——做多的总额等于做空的总额）

    1.2 信号（Signal）是什么¶

    信号是你用来决定 \(w_{i,t}\) 的数值指标。

    一般流程： 1. 计算信号值 \(s_{i,t}\)（比如"过去 12 个月收益率"） 2. 将信号转化为仓位 \(w_{i,t} = f(s_{i,t})\)（比如"信号排名前 20% 的做多，后 20% 的做空"）

    可靠度：Level 1（数学框架本身）。

    2. 动量策略（Momentum）¶

    2.1 核心逻辑¶

    动量（Momentum）：过去涨的资产倾向于继续涨，过去跌的倾向于继续跌。

    这直接违反了弱式 EMH（01-overview），因为你在用历史价格信息预测未来。但大量实证研究发现动量效应确实存在——这是金融学中最知名的异象（anomaly，无法被有效市场理论解释的现象）之一。

    可靠度：Level 2（动量效应的存在性——学术主流认可，Jegadeesh & Titman 1993 是奠基论文）；Level 3（动量的来源——行为偏差还是风险补偿——仍在争论）。

    2.2 两种动量¶

    时间序列动量（TSMOM） 截面动量（XSMOM）
    比较对象 资产和自己的过去比 不同资产之间比
    信号 单一资产过去 \(K\) 个月的收益率 资产在同类中的排名
    做多条件 过去收益率 > 0 过去收益率排名前 20%
    做空条件 过去收益率 < 0 过去收益率排名后 20%
    典型应用 期货、外汇（CTA 基金常用） 股票（因子投资常用）

    2.3 时间序列动量（TSMOM）¶

    信号：过去 \(K\) 个月的累计收益率。

    \[s_{i,t} = \frac{P_{i,t} - P_{i,t-K}}{P_{i,t-K}} = \text{过去 } K \text{ 个月的简单收益率}\]

    仓位规则（最简单版）：

    \[w_{i,t} = \begin{cases} +1 & \text{如果 } s_{i,t} > 0 \text{（过去涨了 → 做多）} \\ -1 & \text{如果 } s_{i,t} < 0 \text{（过去跌了 → 做空）} \end{cases}\]

    回顾期（lookback period）\(K\) 的选择： - 短期（1–4 周）：捕捉短期惯性，但交易成本高 - 中期（3–12 个月）：经典动量的甜蜜区域，文献中最常用 \(K = 12\) 个月 - 长期（3–5 年）：反而变成反转——这不是动量，这是均值回复（下一节）

    2.4 截面动量（XSMOM）¶

    截面动量不看绝对收益率，而是看排名。

    信号：过去 \(K\) 个月收益率在所有股票中的排名。

    仓位规则： 1. 对所有 \(N\) 只股票按过去 12 个月收益率排序 2. 做多排名前 10%–20%（"赢家组合"） 3. 做空排名后 10%–20%（"输家组合"） 4. 这种多空组合叫做 WML（Winners Minus Losers）

    截面动量与多因子模型中的动量因子本质上是同一个东西，只是叫法不同。详见 domains/factor-models/notes/05-classic-factors-1.md。

    2.5 具体数字例子：TSMOM¶

    5 个期货品种过去 12 个月的收益率：

    品种 过去 12 个月收益率 TSMOM 信号 仓位
    原油 +15% 正 做多 (+1)
    黄金 −3% 负 做空 (−1)
    铜 +8% 正 做多 (+1)
    大豆 −12% 负 做空 (−1)
    天然气 +22% 正 做多 (+1)

    假设下个月实际收益率为：原油 +2%、黄金 −1%、铜 −3%、大豆 +5%、天然气 +4%。

    策略收益（等权重，每个品种 1/5 的资金）：

    \[r_{\text{strat}} = \frac{1}{5}[(+1)(+0.02) + (-1)(-0.01) + (+1)(-0.03) + (-1)(+0.05) + (+1)(+0.04)]\]
    \[= \frac{1}{5}[0.02 + 0.01 - 0.03 - 0.05 + 0.04] = \frac{-0.01}{5} = -0.2\%\]

    这个月动量策略亏钱了——铜和大豆的动量失效（铜涨了但反转下跌，大豆跌了但反弹上涨）。动量策略的风险之一就是动量反转（Momentum Crash）。

    3. 均值回复（Mean Reversion）¶

    3.1 核心逻辑¶

    均值回复：价格偏离某个"正常水平"后，会倾向于回归到这个水平。

    和动量完全相反——动量说"涨了会继续涨"，均值回复说"涨太多了会跌回来"。

    类比：橡皮筋——拉得越远，回弹的力越大。

    回忆 03 章的 AR(1) 模型：\(\phi_1 < 0\) 意味着负自相关——上一期高了，这一期倾向于低。这就是均值回复在 AR 模型中的数学表达。

    3.2 什么时候价格会回归均值？¶

    不是所有资产都有均值回复特性。一般来说：

    容易均值回复的 不容易均值回复的
    价差（两只相关股票的价差）——参见 04 章协整 单只股票的价格（价格是随机游走，不回归）
    估值指标（如市盈率）——长期来看公司估值有"合理范围" 短期价格走势（噪声太大）
    波动率——04 章讲了 GARCH 中波动率回归长期水平 处于趋势中的资产（比如牛市中的指数）

    3.3 怎么定义"均值"？¶

    均值定义方式 计算方法 特点
    简单移动平均 过去 \(N\) 天收盘价的平均值 简单，但对新数据反应慢
    指数加权移动平均（EWMA） 给近期数据更高的权重 对新信息反应更快
    协整关系 通过协整伙伴定义"均值" 数学上最严格（04 章详述）

    3.4 Ornstein-Uhlenbeck（OU）过程¶

    均值回复的标准数学模型是 OU 过程：

    \[dX = \theta(\mu - X)dt + \sigma dW\]

    符号解释（逐个）： - \(X\)：当前值（比如价差 \(Z_t\)） - \(dX\)：极短时间内的变化量 - \(\theta > 0\)：回复速率——\(\theta\) 越大，拉回均值的力越强 - \(\mu\)：长期均值——序列围绕这个值波动 - \((\mu - X)\)：当前值和长期均值的偏差 - \(dt\)：极小时间间隔 - \(\sigma\)：波动率 - \(dW\)：布朗运动增量

    读成一句话：变化 = 均值回复的拉力 + 随机波动。

    关键洞察：当 \(X > \mu\) 时，\((\mu - X) < 0\)，所以 \(dX\) 倾向于为负——拉回来。反之亦然。

    和 GBM（03 章）的对比：

    GBM（\(dS = \mu S\, dt + \sigma S\, dW\)） OU（\(dX = \theta(\mu - X)dt + \sigma dW\)）
    漂移 恒定方向（\(\mu S\)） 指向均值（\(\theta(\mu - X)\)）
    长期行为 越走越远 围绕 \(\mu\) 波动
    适合描述 股票价格 利率、价差、波动率

    3.5 OU 过程的半衰期¶

    半衰期（Half-life）：偏差减小到一半所需的时间。

    \[t_{1/2} = \frac{\ln 2}{\theta}\]

    例如：如果 \(\theta = 0.1\)（天），半衰期 \(= \frac{0.693}{0.1} \approx 7\) 天。这意味着价差偏离均值后，大约 7 天会回来一半。

    对策略设计的意义：半衰期决定了你的持仓周期（holding period）——如果半衰期是 7 天，你大约持仓 7–14 天。

    可靠度：Level 1（OU 过程的数学定义）；Level 4（半衰期指导持仓的经验规则）。

    4. 趋势跟踪（Trend Following）¶

    4.1 核心逻辑¶

    趋势跟踪本质上是动量策略的一个子类，但有两个区别： 1. 通常应用于期货市场（而不是股票） 2. 使用技术指标（如均线、通道）来定义趋势，而不是简单的过去收益率

    4.2 移动平均交叉（Moving Average Crossover）¶

    最经典的趋势跟踪信号之一。

    设置：计算两条均线——快速均线（短周期，如 20 日）和慢速均线（长周期，如 60 日）。

    \[\text{MA}_k(t) = \frac{1}{k}\sum_{i=0}^{k-1} P_{t-i}\]

    其中 \(P_{t-i}\) 是第 \(t-i\) 天的收盘价，\(k\) 是均线周期。

    交易规则： - 金叉：快速均线上穿慢速均线 → 做多（\(\text{MA}_{20} > \text{MA}_{60}\)，短期趋势向上超过了长期趋势） - 死叉：快速均线下穿慢速均线 → 做空或平仓

    01-overview 中提到的"20 日均线上穿 60 日均线"就是这个策略。

    4.3 通道突破（Donchian Channel）¶

    唐奇安通道（Donchian Channel）：过去 \(N\) 天的最高价和最低价形成的通道。

    \[\text{Upper}_t = \max(H_{t-1}, H_{t-2}, \ldots, H_{t-N})$$ $$\text{Lower}_t = \min(L_{t-1}, L_{t-2}, \ldots, L_{t-N})\]

    其中 \(H\) 是日最高价，\(L\) 是日最低价（02 章讲的 OHLCV 中的 H 和 L）。

    交易规则： - 价格突破上轨 → 做多（新高 = 上升趋势确认） - 价格跌破下轨 → 做空或平仓

    4.4 CTA 基金¶

    CTA（Commodity Trading Advisor）：专门做趋势跟踪的基金，主要交易期货（商品、外汇、股指、债券期货）。

    CTA 基金的特点： - 在市场剧烈波动（如金融危机）时表现好——因为大趋势明确 - 在市场震荡（没有明确趋势）时表现差——频繁假突破导致亏损 - 与股票市场的相关性低——危机 alpha（Crisis Alpha），在股市暴跌时 CTA 基金往往盈利

    可靠度：Level 2（趋势跟踪策略的有效性——大量实证支持）；Level 4（具体参数选择和 CTA 行业实践——业界经验）。

    5. 动量 vs 均值回复：什么时候用哪个？¶

    5.1 时间尺度是关键¶

    不同时间尺度上，市场呈现不同的特征：

    时间尺度 主导效应 策略类型 原因推测
    日内到几天 均值回复 做市、日内回转 流动性冲击引起的短暂偏离
    1–12 个月 动量 TSMOM、XSMOM 投资者对新信息反应不足
    3–5 年 均值回复 价值投资 长期估值回归

    这意味着：同一个资产可以同时有动量和均值回复特征——取决于你在什么时间尺度上看。

    5.2 对 AR(1) 模型的回顾¶

    回忆 03 章： - \(\phi_1 > 0\)（正自相关）→ 动量 - \(\phi_1 < 0\)（负自相关）→ 均值回复

    如果你对日收益率估计 AR(1)，\(\phi_1\) 的值通常非常接近 0——说明日尺度上的自相关很弱，信号被噪声淹没。这就是为什么量化投资需要大量数据和精密的统计方法。

    可靠度：Level 2（时间尺度与策略类型的对应关系——学术主流观点）；Level 3（原因推测——仍在争论）。

for each trading_day t:
    for each asset i:
        # Step 1: 计算信号
        signal = (price[i][t] - price[i][t - lookback]) / price[i][t - lookback]

        # Step 2: 信号 → 仓位
        if signal > 0:
            weight[i] = +1 / N  # 等权做多，N 是资产总数
        else:
            weight[i] = -1 / N  # 等权做空

    # Step 3: 计算策略收益
    strategy_return[t] = sum(weight[i] * asset_return[i][t] for all i)

    6.2 均值回复策略伪代码（基于协整的配对交易）¶

    # 伪代码：配对交易策略
# 假设已经找到协整的股票对 (A, B)，协整系数 gamma

window = 60  # 用过去 60 天计算均值和标准差

for each trading_day t:
    # Step 1: 计算价差
    spread = price_B[t] - gamma * price_A[t]

    # Step 2: 标准化（z-score）
    spread_mean = mean(spread over past `window` days)
    spread_std = std(spread over past `window` days)
    z_score = (spread - spread_mean) / spread_std

    # Step 3: 交易决策
    if z_score > 2.0:      # 价差太高 → B 相对太贵
        short B, long A     # 做空 B、做多 A
    elif z_score < -2.0:    # 价差太低 → B 相对太便宜
        long B, short A     # 做多 B、做空 A
    elif abs(z_score) < 0.5:  # 价差回归
        close all positions   # 平仓

    这些只是最简单的版本。实际策略还需要考虑： - 交易成本（02 章） - 仓位管理（不是所有资产等权重） - 风险控制（止损、最大仓位限制——07 章详述） - 回测陷阱（前视偏差、存活偏差——08 章详述）

    可靠度：Level 4（具体参数和实现细节——业界经验，不同从业者的选择差异很大）。

    \(s_{i,t} = \frac{P_{i,t} - P_{i,t-K}}{P_{i,t-K}}\) TSMOM 信号 \(K\)=回顾期（常用 12 个月）
