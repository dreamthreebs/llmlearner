    数学工具（下）— 波动率建模与协整¶

    主维度：D2 数学工具 次维度：D3 策略理论（协整是统计套利的数学基础） 关键关系： - GARCH (方法) --推广了--> ARCH (方法) - 协整 (概念) --用于--> 统计套利 (任务) - 波动率聚类 (概念) --依赖--> 条件异方差 (概念)

    前置知识：03-math-tools-1（随机游走、平稳性、AR/MA/ARMA 模型）

    1. 波动率聚类（Volatility Clustering）¶

    1.1 ARMA 的未解之谜¶

    上一章讲的 ARMA 模型假设方差恒定——即噪声 \(\epsilon_t\) 的波动幅度始终一样。但金融数据有一个显著特征打破了这个假设：

    大波动后面跟着大波动，小波动后面跟着小波动。

    这就是波动率聚类（Volatility Clustering）。

    1.2 直觉¶

    想象股票市场： - 2020 年 3 月，新冠疫情爆发，全球股市连续几天暴涨暴跌（每天 ±5% 以上） - 2019 年大部分时间，市场波动很小（每天 ±0.5%）

    波动的幅度不是随机的——平静和剧烈会成团出现。

    用统计语言说：收益率本身 \(r_t\) 可能没有自相关（上一章说的，AR 系数接近 0），但收益率的绝对值 \(|r_t|\) 或平方 \(r_t^2\) 有很强的正自相关。

    这意味着：虽然你不能预测明天价格涨还是跌（方向），但你可以预测明天波动大还是小（幅度）。

    可靠度：Level 1。波动率聚类是金融数据最稳定的经验事实之一（"stylized facts"）。

    2. ARCH 模型¶

    2.1 条件异方差¶

    上一章的 ARMA 模型中，噪声 \(\epsilon_t\) 的方差是固定的 \(\sigma^2\)。条件异方差（Conditional Heteroskedasticity）是说：\(\epsilon_t\) 的方差随时间变化，而且取决于过去的信息。

    分解术语： - 条件（Conditional）：在已知过去信息的条件下 - 异方差（Heteroskedasticity）："hetero"（不同）+ "skedasticity"（方差的分散程度）= 方差不恒定

    2.2 ARCH(1) 模型¶

    ARCH（AutoRegressive Conditional Heteroskedasticity，自回归条件异方差），由 Robert Engle 于 1982 年提出（他因此获得 2003 年诺贝尔经济学奖）。

    模型结构：

    \[\epsilon_t = \sigma_t \cdot z_t, \quad z_t \sim N(0,1)\]
    \[\sigma_t^2 = \omega + \alpha_1 \epsilon_{t-1}^2\]

    符号解释： - \(\epsilon_t\)：收益率中的随机部分（残差） - \(\sigma_t\)：条件标准差——在已知过去信息条件下，\(\epsilon_t\) 的标准差 - \(\sigma_t^2\)：条件方差——波动率的平方 - \(z_t\)：标准正态随机变量（标准化后的噪声） - \(\omega > 0\)：基线方差（保证 \(\sigma_t^2 > 0\)） - \(\alpha_1 \geq 0\)：上一期残差平方的影响系数 - \(\epsilon_{t-1}^2\)：上一期残差的平方——无论上期是大涨还是大跌，平方后都是大的正数

    读成一句话：今天的波动率取决于昨天的残差有多大——如果昨天出现了大的意外（\(\epsilon_{t-1}^2\) 大），今天的波动率也会更大。

    这就解释了波动率聚类！

    可靠度：Level 1。

    3. GARCH(1,1) 模型¶

    3.1 ARCH 的不足¶

    ARCH 模型只看过去一期的残差来预测波动率。但实际中，波动率的持续性很强——一旦进入高波动期，可能持续几周甚至几个月。要用 ARCH 捕捉这种持续性，需要很高的阶数（ARCH(20) 甚至 ARCH(30)），参数太多。

    3.2 GARCH(1,1) 定义¶

    GARCH（Generalized ARCH，广义 ARCH），由 Tim Bollerslev 于 1986 年提出。GARCH(1,1) 是实际应用中最常用的波动率模型：

    \[\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2\]

    符号解释（逐个）： - \(\sigma_t^2\)：今天的条件方差（我们要预测的） - \(\omega > 0\)：长期方差的贡献——即使过去没有任何波动，方差也不会降为 0 - \(\alpha \geq 0\)：冲击系数——昨天的残差平方（意外冲击）对今天方差的影响 - \(\epsilon_{t-1}^2\)：昨天的残差平方（新信息的冲击） - \(\beta \geq 0\)：持续系数——昨天的条件方差对今天方差的影响（波动率的惯性） - \(\sigma_{t-1}^2\)：昨天的条件方差

    读成一句话：今天的波动率 = 基线 + 昨天的意外冲击 + 昨天的波动率水平。

    3.3 参数的直觉¶

    • \(\alpha\) 大：新冲击的影响大——波动率对新消息很敏感
    • \(\beta\) 大：波动率的惯性强——一旦进入高波动期就很难退出
    • \(\alpha + \beta\)：衡量波动率聚类的强度
    • \(\alpha + \beta < 1\)：方差会回归长期水平（平稳条件）
    • \(\alpha + \beta\) 接近 1：方差回归很慢——波动率的记忆很长
    • \(\alpha + \beta = 1\)：IGARCH（积分 GARCH）——方差永远不会回归

    长期方差（也叫无条件方差）：

    \[\sigma_{\text{long}}^2 = \frac{\omega}{1 - \alpha - \beta}\]

    3.4 具体数字例子¶

    设 \(\omega = 0.00001\)，\(\alpha = 0.10\)，\(\beta = 0.85\)。

    长期方差 \(= \frac{0.00001}{1 - 0.10 - 0.85} = \frac{0.00001}{0.05} = 0.0002\) 长期日波动率 \(= \sqrt{0.0002} \approx 0.0141 = 1.41\%\)

    假设昨天的条件方差 \(\sigma_{t-1}^2 = 0.0002\)（处于长期水平），昨天收益率出现大跌 \(\epsilon_{t-1} = -0.04\)（4% 的负冲击）：

    \[\sigma_t^2 = 0.00001 + 0.10 \times (-0.04)^2 + 0.85 \times 0.0002\]
    \[= 0.00001 + 0.10 \times 0.0016 + 0.85 \times 0.0002\]
    \[= 0.00001 + 0.00016 + 0.00017 = 0.00034\]

    今天的日波动率 \(= \sqrt{0.00034} \approx 0.0184 = 1.84\%\)

    对比：长期水平 1.41%，经历大跌后跳升到 1.84%——波动率因为昨天的冲击而升高了。

    接下来，假设今天没有大的冲击（\(\epsilon_t \approx 0\)）：

    \[\sigma_{t+1}^2 = 0.00001 + 0.10 \times 0 + 0.85 \times 0.00034 = 0.00001 + 0.000289 = 0.000299\]

    波动率 \(= \sqrt{0.000299} \approx 0.0173 = 1.73\%\)

    观察：即使今天没有新冲击，波动率仍然较高（1.73%），因为 \(\beta = 0.85\) 的惯性在起作用。波动率会慢慢回到长期水平 1.41%。

    可靠度：Level 1（GARCH 模型本身）；Level 4（具体参数的典型值——来自业界经验，不同市场和资产的参数不同）。

    4. 厚尾分布（Fat Tails）¶

    4.1 正态分布的问题¶

    GBM（03 章）假设收益率服从正态分布。但金融数据的一个核心事实是：

    极端事件发生的频率远高于正态分布的预测。

    具体数字对比：

    事件 正态分布预测的频率 实际观测频率
    日收益率 > 3σ 每 741 天一次（约 3 年） 大约每 1-2 个月
    日收益率 > 5σ 每 350 万天一次（约 1.4 万年） 几乎每年都有
    日收益率 > 10σ 概率 \(\approx 10^{-23}\)（宇宙年龄内不该发生） 1987 年黑色星期一（单日跌 22%）

    厚尾（Fat Tails）的意思是：收益率分布的尾部比正态分布"胖"得多——极端值出现的概率更大。

    4.2 \(t\) 分布作为替代¶

    学生 \(t\) 分布可以更好地拟合金融收益率的厚尾特征。\(t\) 分布有一个额外参数——自由度 \(\nu\)（nu）：

    • \(\nu\) 很大（比如 \(\nu > 30\)）：\(t\) 分布趋近正态分布
    • \(\nu\) 较小（比如 \(\nu = 5\)）：尾部比正态分布厚很多
    • 实际金融数据中，\(\nu\) 通常在 3–8 之间

    你在统计课中学过 \(t\) 分布（用于小样本假设检验），这里是同一个分布，但用在不同的场景——拟合收益率的分布形状。

    4.3 对量化投资的影响¶

    如果你假设收益率是正态分布来做风险管理（比如计算 VaR），你会严重低估极端亏损的概率。这就是为什么 01-overview 提到的 VaR 模型在 2008 年金融危机中大规模失效——模型认为"不可能"的事情发生了。

    可靠度：Level 1（厚尾现象的存在）；Level 2（\(t\) 分布拟合——学术主流但不是唯一选择，也有人用稳定分布等）。

    5. 协整（Cointegration）¶

    5.1 相关性 vs 协整——一个关键区别¶

    你已经知道什么是相关性（Correlation）——两个变量的线性关联程度。但在金融中，相关性和协整是完全不同的概念：

    相关性（Correlation） 协整（Cointegration）
    衡量什么 两个变量短期波动方向的一致性 两个变量长期水平是否保持稳定关系
    是否随时间变化 会显著变化（金融危机时相关性暴增） 通常更稳定
    例子 可口可乐和百事可乐的日收益率高度正相关 可口可乐和百事可乐的股价长期保持稳定比例关系

    5.2 数学定义¶

    假设两个时间序列 \(X_t\) 和 \(Y_t\) 各自都是非平稳的（比如它们是两只股票的价格，都有趋势）。

    如果存在常数 \(\gamma\)，使得它们的线性组合是平稳的：

    \[Z_t = Y_t - \gamma X_t \quad \text{是平稳的}\]

    则称 \(X_t\) 和 \(Y_t\) 是协整的（Cointegrated），\(\gamma\) 叫做协整系数。

    直觉：两只股票各自的价格在"漫步"，但它们之间的价差（经过系数调整后）围绕某个水平上下波动，不会越走越远。就像两个醉汉被一根绳子拴在一起——各自乱走，但不会走太远。

    5.3 Engle-Granger 两步法¶

    Robert Engle（就是提出 ARCH 的那位）和 Clive Granger 提出了检验协整关系的经典方法：

    Step 1：回归

    对 \(Y_t\) 和 \(X_t\) 做普通最小二乘回归（你很熟悉！）：

    \[Y_t = \alpha + \gamma X_t + u_t\]

    得到残差 \(\hat{u}_t = Y_t - \hat{\alpha} - \hat{\gamma} X_t\)。

    Step 2：ADF 检验残差

    对残差 \(\hat{u}_t\) 做 ADF 检验（上一章讲过）： - 如果残差是平稳的 → \(X_t\) 和 \(Y_t\) 协整 - 如果残差是非平稳的 → 没有协整关系

    5.4 具体数字例子¶

    两只同行业的股票价格：

    日期 股票 A (\(X_t\)) 股票 B (\(Y_t\)) 价差 \(Y_t - 1.5 X_t\)
    Day 1 20.0 30.0 0.0
    Day 2 21.0 31.8 0.3
    Day 3 19.5 29.0 −0.25
    Day 4 22.0 33.2 0.2
    Day 5 20.5 30.5 −0.25
    Day 6 23.0 34.8 0.3

    观察：A 和 B 的价格各自在变化（非平稳），但价差 \(Y_t - 1.5 X_t\) 在 0 附近小幅波动（平稳）。这说明 A 和 B 可能是协整的，\(\gamma \approx 1.5\)。

    可靠度：Level 1（协整的数学定义和 Engle-Granger 方法）。

    6. 协整与统计套利¶

    6.1 配对交易的数学基础¶

    配对交易（Pairs Trading）是统计套利中最经典的策略（01-overview 简要提过统计套利），而协整是它的数学基础。

    思路： 1. 找到一对协整的股票 A 和 B 2. 计算价差 \(Z_t = Y_t - \gamma X_t\) 3. 当价差偏离均值太多时交易： - \(Z_t\) 远高于均值 → B 相对于 A "太贵了" → 做空 B、做多 A - \(Z_t\) 远低于均值 → B 相对于 A "太便宜了" → 做多 B、做空 A 4. 等价差回归均值时平仓获利

    "做空"的含义：先借入股票卖出（借的，以后要还），等价格下跌后买回来归还，赚取差价。

    6.2 用标准差定义"偏离太多"¶

    \[Z\text{-score}_t = \frac{Z_t - \mu_Z}{\sigma_Z}\]

    其中 \(\mu_Z\) 和 \(\sigma_Z\) 是价差 \(Z_t\) 的均值和标准差（通常用滚动窗口计算）。

    交易规则（一个简单版本）： - \(|Z\text{-score}| > 2\)：开仓（价差偏离太多） - \(|Z\text{-score}| < 0.5\)：平仓（价差回归了）

    6.3 为什么是协整而不是相关性？¶

    两只高度正相关的股票可能不是协整的——它们可能短期一起涨跌，但长期价差越走越大，永远不会回来。如果你基于相关性做配对交易，可能会血本无归。

    协整保证了一件事：价差长期来看一定会回归。这才是配对交易获利的前提。

    可靠度：Level 1（协整与配对交易的数学关系）；Level 3–4（配对交易在实盘中的盈利能力——学术上有争议，业界经验表明纯价格配对交易的利润空间在缩小）。

    公式速查卡¶

    公式 含义 关键参数
    \(\sigma_t^2 = \omega + \alpha_1 \epsilon_{t-1}^2\) ARCH(1) \(\alpha_1\)=冲击影响
    \(\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2\) GARCH(1,1) \(\alpha\)=冲击，\(\beta\)=惯性，\(\alpha+\beta<1\) 平稳
    \(\sigma_{\text{long}}^2 = \frac{\omega}{1 - \alpha - \beta}\) GARCH 长期方差 \(\alpha+\beta\) 越接近 1，回归越慢
    \(Z_t = Y_t - \gamma X_t\)（平稳） 协整关系 \(\gamma\)=协整系数
    \(Z\text{-score}_t = \frac{Z_t - \mu_Z}{\sigma_Z}\) 标准化价差 配对交易的开平仓信号
