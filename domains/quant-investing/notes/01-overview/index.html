    量化投资全景概览¶

    主维度：全部（跨维度概览） 关键关系： - 量化策略 (方法) --依赖--> 统计信号 (概念) - 策略评估 (方法) --用于--> 策略筛选 (任务) - 回测 (方法) --依赖--> 历史数据 (概念)

    1. 量化投资在干什么¶

    一句话版本¶

    用数学模型和程序代替人脑做投资决策：发现规律 → 构建策略 → 执行交易 → 管理风险。

    一个具体例子¶

    假设你发现一个规律：每次某只股票的 20 日均线（过去 20 天收盘价的平均值）上穿 60 日均线时，接下来一个月涨的概率明显大于跌的概率。

    人工投资者：每天盯盘，肉眼看均线，凭感觉买卖。 量化投资者： 1. 写代码计算均线 2. 在 20 年历史数据上检验这个规律是否真实 3. 计算策略的收益率、风险、最大回撤（从最高点跌到最低点的幅度） 4. 如果统计上显著，写程序自动执行交易

    关键区别不是"用不用电脑"，而是用不用统计方法检验规律是否真实。

    2. 量化投资 vs 其他投资方式¶

    投资方式 决策依据 优势 劣势
    基本面投资 分析公司财务、行业前景 深度理解公司价值 主观判断多、难以规模化
    技术分析 看K线图（用图形表示价格走势的图表）、画趋势线 直观 缺乏统计严谨性
    量化投资 数学模型 + 统计检验 可检验、可复现、纪律性强 容易过拟合、模型失效风险

    可靠度：Level 1。这是教科书级别的分类。

    3. 量化投资的核心流程¶

    数据获取 → 信号研究 → 策略构建 → 回测验证 → 风险管理 → 实盘交易
  ↑                                                        |
  └────────── 策略监控与迭代 ──────────────────────────────┘

    3.1 数据获取¶

    量化投资的原材料。

    数据类型 例子 频率
    行情数据 股价、成交量、买卖盘口（买方和卖方各自挂出的价格和数量） 秒级到日级
    财务数据 营收、利润、资产负债表 季度/年度
    另类数据 卫星图像、新闻情感、社交媒体 不定期

    详见 02 章。

    3.2 信号研究¶

    从数据中发现统计上可重复的规律（alpha 信号）。

    什么是好信号？ - 预测力：信号和未来收益率之间有统计上显著的关系 - 稳定性：不是只在某几年有效 - 经济逻辑：最好有说得通的原因（纯统计规律不够可靠）

    这里"alpha 信号"中的 alpha 和多因子模型域讲的 \(\alpha\) 是同一个概念——因子模型无法解释的超额收益。参见 domains/factor-models/notes/01-overview.md。

    3.3 策略构建¶

    把信号变成具体的交易规则：什么时候买、买多少、什么时候卖。

    主要策略类型（后续章节详细展开）：

    策略类型 核心逻辑 例子
    动量策略 涨的继续涨 买过去 12 个月涨最多的股票
    均值回复 偏离会回来 价格偏离均值太多时反向操作
    统计套利 价差回归 两只相关股票价差偏离时交易
    趋势跟踪 跟随大趋势 均线突破时顺势交易

    详见 05-06 章。

    3.4 回测验证¶

    在历史数据上模拟策略的表现。

    回测的核心难题：过拟合（overfitting）——策略在历史数据上表现很好，但在未来不管用。

    类比你学过的机器学习：这和训练集上效果好、测试集上效果差是完全一样的问题。

    详见 08 章。

    3.5 风险管理¶

    控制亏损的幅度和概率。

    风险度量 含义
    最大回撤（MDD） 从峰值到谷底的最大跌幅——你能忍受多大的亏损？
    VaR（Value at Risk，在险价值） 在 95%/99% 置信度下的最大可能亏损
    夏普比率 每单位风险获得的超额收益（参见 domains/factor-models/notes/01-overview.md）

    详见 07 章。

    3.6 实盘交易¶

    把策略从回测环境搬到真实市场。中间有很多工程问题：

    • 滑点（slippage）：你想以 10 元买入，但实际成交价是 10.02 元——买入价格和预期价格之间的差异
    • 交易成本：手续费、印花税、冲击成本（你的大单本身会推动价格）
    • 延迟：从发出信号到实际成交的时间差

    详见 09 章。

    4. 有效市场假说——量化投资的"对立面"¶

    有效市场假说（Efficient Market Hypothesis, EMH）由 Eugene Fama（就是 Fama-French 三因子模型的那个 Fama）提出，核心主张：

    资产价格已经反映了所有可获得的信息，因此不可能持续获得超额收益。

    三种形式：

    形式 价格反映了什么信息 哪些策略失效
    弱式有效 所有历史价格信息 技术分析（看历史价格预测未来）失效
    半强式有效 所有公开信息 基本面分析也失效
    强式有效 所有信息（含内幕） 任何策略都失效——即使有内幕消息

    量化投资的立场：市场大部分是有效的，但存在局部和暂时的无效，可以被系统性地利用。

    可靠度：Level 2。EMH 是学术主流框架，但"市场到底有多有效"是持续争论的问题。

    5. 知识维度与学习路径¶

    五个维度¶

    维度 核心问题 你已有的基础
    D1 市场与数据 市场怎么运作？数据从哪来？ 需要从头学
    D2 数学工具 怎么用数学描述价格运动？ 概率论/回归已有，随机过程需补
    D3 策略理论 怎么找信号、建策略？ 需要从头学
    D4 风险与评估 策略是真好还是假好？ 统计检验基础已有
    D5 实盘工程 怎么从回测到实盘？ Python 熟练，框架需要学

    学习路径¶

    概览（本章）→ 市场与数据 → 数学工具 → 策略理论 → 风险评估 → 回测 → 实盘

    6. 与多因子模型域的关系¶

    多因子模型域 量化投资域
    侧重资产定价理论 侧重策略开发方法论
    核心问题：为什么不同股票收益率不同？ 核心问题：怎么找到可重复的交易信号？
    CAPM、APT、Fama-French 动量、均值回复、统计套利
    因子 = 风险补偿（学术视角） 因子 = 交易信号（实操视角）

    两个域互补：多因子模型提供"因子是什么"的理论基础，本域讲"怎么用因子和其他信号赚钱"。

    公式速查卡¶

    公式 含义
    \(\text{简单收益率} = \frac{P_t - P_{t-1}}{P_{t-1}}\) 一期的价格变化比率
    \(\text{Sharpe} = \frac{E[r] - r_f}{\sigma}\) 每单位风险的超额收益（参见多因子域）
    \(\text{MDD} = \max_{t} \left(\frac{\text{peak}_t - P_t}{\text{peak}_t}\right)\) 最大回撤：从历史峰值到谷底的最大跌幅比例
