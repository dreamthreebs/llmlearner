## 1. 随机游走（Random Walk）

### 1.1 直觉

想象一个醉汉在一条直路上走。每一步他等概率地向前走一步或向后走一步。你能预测他下一步的位置吗？**不能**——你只知道他现在在哪，但下一步是完全随机的。

这就是随机游走的核心思想：**未来的位置只取决于当前位置加上一个随机扰动**。

### 1.2 数学定义

$$P_t = P_{t-1} + \epsilon_t, \quad \epsilon_t \sim \text{i.i.d.}(0, \sigma^2)$$

符号解释：
- $P_t$：第 $t$ 个时刻的价格
- $P_{t-1}$：前一个时刻的价格
- $\epsilon_t$：随机扰动（噪声），均值为 0，方差为 $\sigma^2$
- i.i.d.：独立同分布（Independent and Identically Distributed）——每一步的随机扰动互不影响，且来自同一个分布

### 1.3 随机游走与 EMH 的关系

01-overview 中讲过**有效市场假说（EMH）**：价格已经反映了所有信息。

如果 EMH 成立（弱式），那么过去的价格信息无法预测未来价格，价格变化只由新信息驱动——而新信息的到来是随机的。这正好就是随机游走！

$$\text{EMH（弱式）} \implies \text{价格服从随机游走}$$

反过来说，如果你发现价格**不**是随机游走（比如有动量效应——涨了还会继续涨），就说明市场至少不是弱式有效的，存在可以利用的规律。

> **可靠度**：Level 1（随机游走的数学定义）；Level 2（EMH 与随机游走的对应关系——学术主流但有争议细节）。

### 1.4 具体数字例子

假设某股票今天价格 $P_0 = 100$ 元，每日波动 $\epsilon_t \sim N(0, 4)$（标准差 $\sigma = 2$ 元）。

| 天 | $\epsilon_t$ | $P_t = P_{t-1} + \epsilon_t$ |
|----|-------------|------|
| 0  | — | 100.00 |
| 1  | +1.3 | 101.30 |
| 2  | −0.8 | 100.50 |
| 3  | +2.1 | 102.60 |
| 4  | −1.5 | 101.10 |
| 5  | −0.3 | 100.80 |

关键观察：$P_t$ 的变化没有规律（随机的），但 $P_t$ 本身在 100 附近"漫步"。

## 2. 布朗运动与几何布朗运动

### 2.1 从随机游走到布朗运动

随机游走是**离散时间**的模型（一步一步走）。如果我们让步长趋近于 0、步数趋近于无穷，就得到**连续时间**版本——**布朗运动（Brownian Motion）**，也叫**维纳过程（Wiener Process）**。

布朗运动 $W_t$ 的性质：
1. $W_0 = 0$
2. **独立增量**：$W_t - W_s$（$t > s$）与过去的路径无关
3. **正态增量**：$W_t - W_s \sim N(0, t - s)$
4. 路径是连续的（没有跳跃）

直觉：布朗运动就是"无限细密的随机游走"。

### 2.2 几何布朗运动（GBM）

直接用布朗运动来描述股票价格有一个问题：布朗运动可以取负值，但股票价格不能为负数。

解决方案：让**价格的变化率**（而不是价格本身）服从布朗运动。这就是**几何布朗运动（Geometric Brownian Motion, GBM）**：

$$dS = \mu S \, dt + \sigma S \, dW$$

符号解释（**逐个**）：
- $S$：当前股票价格
- $dS$：价格在极短时间 $dt$ 内的变化量
- $\mu$：**漂移率（Drift）**——价格的平均增长率。$\mu > 0$ 表示价格倾向于上涨
- $\sigma$：**波动率（Volatility）**——价格波动的幅度。$\sigma$ 越大，价格波动越剧烈
- $dt$：极小的时间间隔
- $dW$：布朗运动的增量，$dW \sim N(0, dt)$

把整个公式读成一句话：**价格的变化 = 确定性的趋势部分 + 随机波动部分**。

更直观的离散形式（方便数字计算）：

$$\frac{\Delta S}{S} \approx \mu \Delta t + \sigma \sqrt{\Delta t} \cdot z, \quad z \sim N(0,1)$$

即：**收益率 ≈ 趋势 + 随机噪声**。

### 2.3 具体数字例子

假设 $S_0 = 100$，$\mu = 0.10$（年化 10% 增长），$\sigma = 0.20$（年化 20% 波动率），$\Delta t = 1/252$（一个交易日，一年约 252 个交易日）。

$$\frac{\Delta S}{S} \approx 0.10 \times \frac{1}{252} + 0.20 \times \sqrt{\frac{1}{252}} \cdot z$$

$$\frac{\Delta S}{S} \approx 0.000397 + 0.01260 \cdot z$$

其中 $z \sim N(0,1)$。

趋势部分：每天约 +0.04%（非常小）
波动部分：标准差约 1.26%（远大于趋势！）

这意味着：**日尺度上，噪声远大于信号**。这就是为什么量化投资者需要大量数据和统计方法才能从噪声中提取信号。

> **可靠度**：Level 1（GBM 的数学定义）；Level 2（GBM 作为股价模型——它是 Black-Scholes 期权定价的基础，但实际股价分布比 GBM 假设的更"厚尾"，详见 04 章）。

## 3. 平稳性（Stationarity）

### 3.1 为什么平稳性重要

**问题**：你能对股票价格直接做回归/建模吗？

答案是**不能**——或者说，直接建模效果很差。原因在于价格是**非平稳的（Non-stationary）**。

**直觉**：一个平稳的过程就像一个围绕某个水平线上下波动的信号——它的统计特征（均值、方差）不随时间改变。而股票价格明显不是这样——它可以从 10 元涨到 1000 元。

### 3.2 严格定义

**（弱）平稳性**要求：
1. **均值恒定**：$E[y_t] = \mu$，对所有 $t$ 相同
2. **方差恒定**：$\text{Var}(y_t) = \sigma^2$，对所有 $t$ 相同
3. **自协方差只取决于时间差**：$\text{Cov}(y_t, y_{t+k})$ 只依赖于 $k$，不依赖于 $t$

### 3.3 价格 vs 收益率

| 性质 | 价格 $P_t$ | 收益率 $r_t = \frac{P_t - P_{t-1}}{P_{t-1}}$ |
|------|-----------|------|
| 均值恒定？ | ❌ 价格有明显趋势 | ✅ 通常在 0 附近波动 |
| 方差恒定？ | ❌ 波动幅度随价格水平变化 | ⚠️ 大致恒定（但有波动率聚类，见 04 章） |
| 平稳？ | ❌ | ✅（近似） |

结论：**量化投资中几乎所有的建模都在收益率上做，而不是价格上做**。

### 3.4 ADF 检验

怎么用统计方法判断一个序列是否平稳？最常用的是 **ADF 检验（Augmented Dickey-Fuller Test）**。

基本思想：检验时间序列是否有**单位根（Unit Root）**。有单位根 = 非平稳。

- **原假设 $H_0$**：序列有单位根（非平稳）
- **备择假设 $H_1$**：序列没有单位根（平稳）
- p-value 小（通常 < 0.05）→ 拒绝 $H_0$ → 序列是平稳的

Python 中用 `statsmodels` 一行代码搞定：

```python
from statsmodels.tsa.stattools import adfuller
result = adfuller(returns_series)
print(f'ADF Statistic: {result[0]:.4f}, p-value: {result[1]:.4f}')
```

> **可靠度**：Level 1。

## 4. 自回归模型（AR）

### 4.1 核心思想

**自回归（AutoRegressive, AR）**的"自"是指**用自己的过去值来预测自己**。

**AR(1) 模型**：

$$y_t = c + \phi_1 y_{t-1} + \epsilon_t$$

符号解释：
- $y_t$：当前时刻的值（注意：在量化投资中通常是**收益率**，不是价格）
- $c$：常数项（截距）
- $\phi_1$：**自回归系数**——$y_{t-1}$ 对 $y_t$ 的影响程度
- $y_{t-1}$：上一个时刻的值
- $\epsilon_t$：随机噪声，$\epsilon_t \sim \text{i.i.d.}(0, \sigma^2)$

和随机游走对比：
- 随机游走：$P_t = P_{t-1} + \epsilon_t$（$\phi_1 = 1$，对价格建模）
- AR(1)：$y_t = c + \phi_1 y_{t-1} + \epsilon_t$（$|\phi_1| < 1$，对收益率建模）

**关键区别**：当 $|\phi_1| < 1$ 时，AR(1) 是平稳的——序列会围绕均值 $\frac{c}{1 - \phi_1}$ 上下波动。

### 4.2 $\phi_1$ 的含义

- $\phi_1 > 0$：**正自相关**——上一期高，这一期也倾向于高 → 类似动量
- $\phi_1 < 0$：**负自相关**——上一期高，这一期倾向于低 → 类似均值回复
- $\phi_1 = 0$：白噪声（White Noise）——完全随机，无法预测
- $|\phi_1| < 1$：平稳条件——偏离均值后会"拉回来"

### 4.3 具体数字例子

设 $c = 0.001$，$\phi_1 = 0.3$，$\sigma = 0.01$。长期均值 $= \frac{0.001}{1 - 0.3} \approx 0.00143$。

| $t$ | $y_{t-1}$ | $\epsilon_t$ | $y_t = 0.001 + 0.3 \times y_{t-1} + \epsilon_t$ |
|-----|----------|-------------|------|
| 1 | 0.00143 (从均值出发) | +0.005 | 0.001 + 0.3 × 0.00143 + 0.005 = 0.00643 |
| 2 | 0.00643 | −0.008 | 0.001 + 0.3 × 0.00643 − 0.008 = −0.00507 |
| 3 | −0.00507 | +0.003 | 0.001 + 0.3 × (−0.00507) + 0.003 = 0.00248 |
| 4 | 0.00248 | +0.001 | 0.001 + 0.3 × 0.00248 + 0.001 = 0.00274 |

观察：序列在 0 附近波动（平稳），$\phi_1 = 0.3$ 的正自相关让序列有轻微的惯性但不会发散。

### 4.4 AR(p) 推广

AR(1) 只用一步过去值。**AR(p)** 用过去 $p$ 步：

$$y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \cdots + \phi_p y_{t-p} + \epsilon_t$$

"$p$"叫做**阶数（order）**，通过信息准则（如 AIC、BIC——你在统计课应该学过）来选择。

> **可靠度**：Level 1。

## 5. 移动平均模型（MA）

### 5.1 注意：此"移动平均"非彼"均线"！

在量化投资中，"移动平均"有两个完全不同的含义：
- **均线（Moving Average of Price）**：过去 $N$ 天收盘价的平均值——这是信号/指标（01-overview 提到的 20 日均线）
- **MA 模型（Moving Average Model）**：时间序列模型——本节要讲的内容

**两者没有直接关系**，名字撞了而已。

### 5.2 MA(1) 模型

$$y_t = c + \epsilon_t + \theta_1 \epsilon_{t-1}$$

符号解释：
- $y_t$：当前时刻的值
- $c$：常数项
- $\epsilon_t$：当前的随机冲击
- $\theta_1$：**MA 系数**——上一期随机冲击对当前值的影响
- $\epsilon_{t-1}$：上一期的随机冲击

和 AR(1) 的对比：

| 模型 | 当前值取决于 | 直觉 |
|------|-----------|------|
| AR(1) | 上一期的**观测值** $y_{t-1}$ | "我的过去决定我的现在" |
| MA(1) | 上一期的**随机冲击** $\epsilon_{t-1}$ | "过去的意外事件影响我" |

### 5.3 具体数字例子

设 $c = 0.002$，$\theta_1 = 0.5$，$\sigma = 0.01$。

| $t$ | $\epsilon_t$ | $\epsilon_{t-1}$ | $y_t = 0.002 + \epsilon_t + 0.5 \times \epsilon_{t-1}$ |
|-----|-------------|-----------------|------|
| 1 | +0.008 | 0 (初始) | 0.002 + 0.008 + 0 = 0.010 |
| 2 | −0.005 | +0.008 | 0.002 − 0.005 + 0.5 × 0.008 = 0.001 |
| 3 | +0.012 | −0.005 | 0.002 + 0.012 + 0.5 × (−0.005) = 0.0115 |
| 4 | −0.003 | +0.012 | 0.002 − 0.003 + 0.5 × 0.012 = 0.005 |

观察：$t=1$ 时的大冲击 $\epsilon_1 = +0.008$ 在 $t=2$ 时仍然有影响（通过 $0.5 \times 0.008 = 0.004$），但到 $t=3$ 时就完全消失了。MA(1) 的"记忆"只有 1 步。

### 5.4 MA(q) 推广

$$y_t = c + \epsilon_t + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \cdots + \theta_q \epsilon_{t-q}$$

**MA(q) 模型的记忆恰好是 $q$ 步**：$q$ 步之前的冲击对现在没有任何影响。

> **可靠度**：Level 1。

## 6. ARMA 模型

### 6.1 组合 AR 和 MA

**ARMA(p,q)** 就是把 AR(p) 和 MA(q) 合在一起：

$$y_t = c + \underbrace{\phi_1 y_{t-1} + \cdots + \phi_p y_{t-p}}_{\text{AR 部分}} + \epsilon_t + \underbrace{\theta_1 \epsilon_{t-1} + \cdots + \theta_q \epsilon_{t-q}}_{\text{MA 部分}}$$

直觉：序列的当前值既受自己过去值的影响（AR），也受过去随机冲击的影响（MA）。

### 6.2 为什么需要 ARMA？

实际的金融时间序列往往既有 AR 成分又有 MA 成分。用纯 AR 或纯 MA 可能需要很高的阶数，而 ARMA 用较少的参数就能捕捉同样的动态。

**模型选择**（你已经熟悉的套路）：
1. 用 ADF 检验确认序列是否平稳
2. 如果不平稳 → 取差分（一阶差分：$\Delta y_t = y_t - y_{t-1}$）→ 得到 **ARIMA** 模型（I 表示 Integrated——差分次数）
3. 用 AIC/BIC 选择 $p$ 和 $q$
4. 检验残差是否为白噪声（Ljung-Box 检验——检测残差中是否还有自相关结构）

### 6.3 ARMA 在量化投资中的局限

ARMA 假设**方差恒定**，但金融数据有一个显著特征：**波动率聚类**——大波动后面往往跟着大波动。这违反了 ARMA 的假设。

解决方案：GARCH 模型（下一章的主题）。

> **可靠度**：Level 1（ARMA 模型本身）；Level 2（ARMA 在金融中的适用性评估）。

| 公式 | 含义 | 关键参数 |
|------|------|---------|

| $y_t = c + \epsilon_t + \theta_1 \epsilon_{t-1}$ | MA(1) | 记忆只有 1 步 |
| AR(1) 长期均值 $= \frac{c}{1 - \phi_1}$ | AR(1) 均值回归的中心 | $|\phi_1|<1$ 时成立 |
