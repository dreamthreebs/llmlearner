# 分析哲学术语表

| 缩写/术语 | 全称/原语 | 简述 |
|-----------|----------|------|
| a posteriori | 后验 | 需要经验才能知道的知识 |
| a priori | 先验 | 不依赖于经验就能知道的知识 |
| analytic | 分析的 | 命题仅凭其含义就为真（如"单身汉是未婚男性"） |
| argument | 论证 | 由前提和结论组成的推理结构 |
| Austin | 奥斯汀 | 日常语言哲学家，言语行为理论创始人 |
| Ayer | 艾耶尔 | 逻辑实证主义代表，著《语言、真理与逻辑》 |
| behaviorism | 行为主义 | 心灵状态就是行为倾向（赖尔） |
| category mistake | 范畴错误 | 把属于一个范畴的概念当作另一个范畴来理解 |
| Chalmers | 查尔默斯 | 提出意识的"困难问题"（hard problem） |
| Chinese Room | 中文房间 | 塞尔反对强 AI 的思想实验 |
| coherence theory | 融贯论 | 真理 = 与其他信念的融贯性 |
| compatibilism | 相容论 | 自由意志和决定论可以兼容 |
| continental philosophy | 大陆哲学 | 与分析哲学对立的传统（胡塞尔、海德格尔、德里达等） |
| conversational implicature | 会话含义 | 格赖斯理论：话语的言外之意 |
| cooperative principle | 合作原则 | 格赖斯：交流参与者默认的合作态度 |
| correspondence theory | 符合论 | 真理 = 命题与事实的符合 |
| counterfactual | 反事实条件句 | "如果 A 没发生，B 就不会发生" |
| definite description | 确定描述 | "the X that is Y"形式的表达（罗素） |
| deflationism | 紧缩论 | "真"不添加实质内容（"p 是真的" = p） |
| determinism | 决定论 | 一切事件由先前状态和自然规律完全决定 |
| dualism | 二元论 | 心灵和身体是两种不同实体（笛卡尔） |
| emotivism | 情感主义 | 道德判断只是情感的表达（Ayer） |
| epistemology | 认识论 | 研究知识的本质、来源和范围 |
| experimental philosophy | 实验哲学 | 用实证方法（问卷、实验）测试哲学直觉 |
| family resemblance | 家族相似 | 维特根斯坦：概念之间只有重叠的相似而非共同本质 |
| Frege | 弗雷格 | Gottlob Frege (1848-1925)，现代逻辑和语言哲学之父 |
| functionalism | 功能主义 | 心灵状态由其功能角色定义（输入-输出-关系） |
| Gettier problem | 盖梯尔问题 | 对 JTB 知识定义的反例 |
| Grice | 格赖斯 | H.P. Grice，提出会话含义理论 |
| hard problem | 困难问题 | 为什么物理过程伴随主观体验？（查尔默斯） |
| identity theory | 同一论 | 心灵状态就是脑状态 |
| illocutionary act | 言外行为 | 说话的目的（命令、承诺、提问） |
| JTB | 被证成的真信念 | Justified True Belief，知识的传统定义 |
| Kripke | 克里普克 | Saul Kripke，《命名与必然性》 |
| language game | 语言游戏 | 后期维特根斯坦：语言在特定活动中的使用 |
| Lewis | 刘易斯 | David Lewis，模态实在论和反事实条件句理论 |
| locutionary act | 言内行为 | 说出语句本身 |
| logical positivism | 逻辑实证主义 | 维也纳学派，证实原则 |
| meaning is use | 意义即使用 | 后期维特根斯坦的核心主张 |
| meta-ethics | 元伦理学 | 研究道德判断的本质（是事实还是情感？） |
| metaphysics | 形而上学 | 研究存在的基本结构（什么存在？怎么存在？） |
| modal realism | 模态实在论 | 所有可能世界都真实存在（Lewis） |
| modus ponens | 肯定前件 | 有效论证形式：如果 P 则 Q；P；所以 Q |
| modus tollens | 否定后件 | 有效论证形式：如果 P 则 Q；非 Q；所以非 P |
| moral realism | 道德实在论 | 存在客观的道德事实 |
| multiple realizability | 多重可实现性 | 同一心灵状态可在不同物理系统中实现 |
| normative ethics | 规范伦理学 | 什么是对的？什么是错的？ |
| Parfit | 帕菲特 | Derek Parfit，人格同一性理论 |
| perlocutionary act | 言后行为 | 说话的实际效果 |
| picture theory | 图像论 | 早期维特根斯坦：命题是事实的逻辑图像 |
| possible worlds | 可能世界 | 理解必然性和可能性的理论工具 |
| predicate logic | 谓词逻辑 | 弗雷格发明的逻辑系统，可表达"所有"和"存在" |
| prescriptivism | 规约主义 | 道德判断是普遍化的命令（Hare） |
| private language | 私有语言 | 只有一个人能理解的语言（维特根斯坦论证其不可能） |
| proposition | 命题 | 可以为真或为假的陈述 |
| propositional logic | 命题逻辑 | 用逻辑联结词连接命题的逻辑系统 |
| qualia | 感质 | 主观经验的质——"看到红色时的那种感觉" |
| Quine | 奎因 | W.V.O. Quine，翻译不确定性、反分析/综合区分 |
| Rawls | 罗尔斯 | John Rawls，《正义论》 |
| reference | 指称 | 表达式所指向的对象（弗雷格） |
| reliabilism | 可靠主义 | 知识来自可靠的认知过程 |
| rigid designator | 固定指示词 | 在所有可能世界中指同一对象的表达（克里普克） |
| Russell | 罗素 | Bertrand Russell (1872-1970)，确定描述理论 |
| Ryle | 赖尔 | Gilbert Ryle，《心的概念》 |
| Searle | 塞尔 | John Searle，言语行为理论、中文房间论证 |
| sense | 涵义 | 表达式的呈现方式（弗雷格） |
| sound | 健全的 | 论证有效且所有前提为真 |
| speech act | 言语行为 | 说话本身就是一种行为（奥斯汀） |
| synthetic | 综合的 | 命题的真假取决于世界怎么样 |
| Tractatus | 《逻辑哲学论》 | 维特根斯坦早期代表作（1921） |
| truth table | 真值表 | 列出所有输入组合下的真假值 |
| valid | 有效的 | 如果所有前提为真则结论必然为真 |
| veil of ignorance | 无知之幕 | 罗尔斯正义论的核心思想实验 |
| verification principle | 证实原则 | 命题有意义当且仅当可被经验验证 |
| Vienna Circle | 维也纳学派 | 逻辑实证主义的学术团体 |
| Wittgenstein | 维特根斯坦 | Ludwig Wittgenstein (1889-1951)，两个阶段 |
