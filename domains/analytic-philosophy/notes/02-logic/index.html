    逻辑基础 — 命题逻辑、谓词逻辑、论证有效性¶

    主维度：D1 历史与方法 + D2 逻辑与语言 关键关系： - 命题逻辑 (方法) --依赖--> 论证 (概念) - 谓词逻辑 (方法) --推广了--> 命题逻辑 (方法) - 谓词逻辑 (方法) --依赖--> 弗雷格 (概念)

    1. 为什么需要形式逻辑¶

    在 01 章我们说过，分析哲学强调论证的清晰性。但自然语言天生有两个毛病：

    歧义（ambiguity）——同一个词或句子可以有多种意思。

    例子："小明在银行旁边看到了她用望远镜"。 - 谁用望远镜？小明还是她？ - "银行"是金融机构还是河岸？ - 同一个句子至少有四种理解。

    含混（vagueness）——概念的边界不清楚。

    例子："这个人很高"。180cm 算高吗？175cm 呢？没有一个精确的分界线。

    形式逻辑（formal logic）的目的就是：把论证的结构从自然语言中提取出来，用符号精确表达，让歧义和含混无处藏身。

    它并不是要取代自然语言（我们日常交流不可能全用符号），而是提供一个检验工具：当你不确定一个论证到底对不对的时候，把它翻译成形式语言，结构上的错误就一目了然。

    可靠度：Level 1。形式逻辑作为分析工具是学术共识。

    2. 命题逻辑¶

    2.1 什么是命题¶

    命题（proposition）是一个可以判断真假的陈述。

    • "北京是中国的首都" → 这是一个命题（而且为真）
    • "2 + 3 = 6" → 这是一个命题（为假）
    • "把门关上！" → 这不是命题（命令没有真假）
    • "今天天气好吗？" → 这不是命题（问句没有真假）

    命题逻辑不关心命题内部的结构（谁对谁做了什么），只关心命题之间的逻辑关系——比如"如果 A 那么 B"、"A 而且 B"。

    我们用大写字母 P、Q、R 等来代表命题。比如： - P = "今天下雨" - Q = "我带伞"

    2.2 逻辑联结词¶

    逻辑联结词（logical connective）是把命题组合成更复杂命题的符号。一共有五个最基本的：

    ¬（非，negation）¶

    ¬P 表示"P 不成立"。如果 P 为真，¬P 为假；如果 P 为假，¬P 为真。

    P = "今天下雨" ¬P = "今天不下雨"

    ∧（与，conjunction）¶

    P ∧ Q 表示"P 和 Q 都成立"。只有两个都为真时，P ∧ Q 才为真。

    P = "今天下雨"，Q = "我带伞" P ∧ Q = "今天下雨，而且我带了伞" 只有确实下雨了、也确实带了伞，这个组合才为真。

    ∨（或，disjunction）¶

    P ∨ Q 表示"P 和 Q 至少有一个成立"。只要有一个为真就为真，两个都假才为假。

    P ∨ Q = "今天下雨，或者我带了伞（或者两者都是）"

    注意：逻辑中的"或"是包含性的——"P 或 Q"也包括"P 和 Q 都成立"的情况。这和日常用语不同——"你要咖啡还是茶？"通常暗示你只能选一个，但逻辑的 ∨ 不排斥"两个都要"。

    →（如果…那么…，conditional / 条件式）¶

    P → Q 表示"如果 P，那么 Q"。

    这是最容易混淆的一个。它的含义是：只有在 P 为真而 Q 为假时，P → Q 才为假。其他情况都算真。

    直觉上的理解：P → Q 是一个承诺——"如果 P 发生了，我保证 Q 也会发生"。这个承诺什么时候被违背？只有一种情况：P 确实发生了，但 Q 没有。

    P Q P → Q
    真 真 真（承诺没被违背）
    真 假 假（承诺被违背了！）
    假 真 真（P 没发生，承诺没被考验）
    假 假 真（P 没发生，承诺没被考验）

    例子：老师说"如果你考满分，我就请你吃饭"。 - 你考了满分，老师请了你 → 承诺兑现（真） - 你考了满分，老师没请你 → 承诺违背（假） - 你没考满分，不管老师请不请你 → 承诺没有被考验，不算违背（真）

    P → Q 中的 P 叫做前件（antecedent），Q 叫做后件（consequent）。

    ↔（当且仅当，biconditional）¶

    P ↔ Q 表示"P 和 Q 要么同时为真，要么同时为假"。等价于 (P → Q) ∧ (Q → P)。

    P ↔ Q = "当且仅当今天下雨，我才带伞" 意思是：下雨就一定带伞，不下雨就一定不带伞。

    2.3 真值表¶

    真值表（truth table）是一种穷举所有可能组合来判断复合命题真假的方法。

    比如，我们要验证 ¬(P ∧ Q) 是否等价于 ¬P ∨ ¬Q（德摩根定律）：

    P Q P ∧ Q ¬(P ∧ Q) ¬P ¬Q ¬P ∨ ¬Q
    真 真 真 假 假 假 假
    真 假 假 真 假 真 真
    假 真 假 真 真 假 真
    假 假 假 真 真 真 真

    ¬(P ∧ Q) 这一列和 ¬P ∨ ¬Q 这一列完全一样，所以两个表达式是逻辑等价的——在任何情况下都有相同的真假值。

    用日常语言说："并非（P 且 Q）"等同于"非 P 或者非 Q"。想想也对——要否定"两个都成立"，只要其中一个不成立就够了。

    可靠度：Level 1。命题逻辑及真值表方法是 19 世纪以来的数学/逻辑学共识。

    3. 论证的有效性¶

    01 章我们定义了：一个论证是有效的（valid），当且仅当如果所有前提为真，结论必然为真。

    现在我们有了形式工具，可以精确地检验有效性了。

    3.1 经典的有效论证形式¶

    肯定前件（modus ponens，MP）¶

    前提 1：P → Q（如果 P，那么 Q）
前提 2：P（P 为真）
结论：Q（所以 Q 为真）

    例子： 前提 1：如果下雨，地面会湿 前提 2：现在下雨了 结论：所以地面会湿

    这是最直观的推理形式。前件成立 → 后件必然成立。

    否定后件（modus tollens，MT）¶

    前提 1：P → Q（如果 P，那么 Q）
前提 2：¬Q（Q 为假）
结论：¬P（所以 P 为假）

    例子： 前提 1：如果下雨，地面会湿 前提 2：地面没有湿 结论：所以没有下雨

    为什么这个有效？因为如果 P 为真，Q 就必须为真（根据前提 1）。但 Q 为假（前提 2），所以 P 不可能为真。

    3.2 经典的无效论证（谬误）¶

    以下两种论证形式看起来合理，但逻辑上无效，在日常争论中极其常见。

    肯定后件谬误（affirming the consequent）¶

    前提 1：P → Q（如果 P，那么 Q）
前提 2：Q（Q 为真）
结论：P（所以 P 为真）  ← 无效！

    例子： 前提 1：如果下雨，地面会湿 前提 2：地面湿了 结论：所以下雨了 ← 错！地面可能因为洒水车经过而湿

    问题在于：P → Q 只说"P 是 Q 的一个充分条件"，不是说 P 是让 Q 为真的唯一原因。

    否定前件谬误（denying the antecedent）¶

    前提 1：P → Q（如果 P，那么 Q）
前提 2：¬P（P 为假）
结论：¬Q（所以 Q 为假）  ← 无效！

    例子： 前提 1：如果下雨，地面会湿 前提 2：没有下雨 结论：所以地面不湿 ← 错！可能有其他原因让地面湿

    3.3 为什么识别谬误很重要¶

    日常论证中大量出现这两种谬误：

    "如果一个人受过高等教育，他就有道德素养。他没受过高等教育，所以他没有道德素养。"（否定前件谬误）

    "如果某种药有效，病人的症状会缓解。这个病人的症状缓解了，所以这种药有效。"（肯定后件谬误——症状可能自然缓解了）

    分析哲学之所以重视形式逻辑，一个重要原因就是：很多哲学争论中的分歧，不是因为人们对世界的看法不同，而是因为论证本身有逻辑错误。形式化可以让这些错误暴露出来。

    可靠度：Level 1。有效/无效论证形式是逻辑学教科书共识。

    4. 谓词逻辑¶

    4.1 命题逻辑的局限¶

    命题逻辑把每个命题当作不可分割的整体（用 P、Q 代表）。但有些论证的有效性取决于命题的内部结构，命题逻辑无法处理：

    前提 1：所有人都会死
前提 2：苏格拉底是人
结论：苏格拉底会死

    在命题逻辑中，这三句话只是 P、Q、R——三个没有内部关系的命题。你看不出为什么结论能从前提推出。

    要处理"所有""存在"这类概念，我们需要谓词逻辑（predicate logic，也叫一阶逻辑 first-order logic）。

    4.2 谓词与个体¶

    个体（individual）是我们谈论的对象，用小写字母表示：a、b、c …

    a = 苏格拉底，b = 柏拉图

    谓词（predicate）描述个体的性质或个体之间的关系，用大写字母表示：

    H(x) = "x 是人" M(x) = "x 会死" L(x, y) = "x 爱 y"

    H(a) 就表示"苏格拉底是人"，L(a, b) 就表示"苏格拉底爱柏拉图"。

    4.3 量词¶

    谓词逻辑最关键的发明是两个量词（quantifier）：

    ∀（全称量词，universal quantifier）¶

    ∀x 读作"对所有 x"。

    ∀x(H(x) → M(x)) = "对所有 x，如果 x 是人，那么 x 会死" = "所有人都会死"

    ∃（存在量词，existential quantifier）¶

    ∃x 读作"存在某个 x"。

    设 W(x) = "x 是聪明的"，则： ∃x(H(x) ∧ W(x)) = "存在某个 x，x 是人而且 x 是聪明的" = "有些人是聪明的"

    注意一个细节：∀ 通常和 → 搭配（"对所有 x，如果 x 是……那么……"），∃ 通常和 ∧ 搭配（"存在某个 x，x 是……而且……"）。如果搞反了，意思会完全不同：

    • ∀x(H(x) ∧ M(x)) 的意思是"所有东西都是人而且都会死"——这显然不是我们想说的
    • ∃x(H(x) → M(x)) 的意思比较怪，因为只要存在一个非人的东西 x，整个命题就为真（回忆一下条件式 →：前件为假时整个条件式为真）

    4.4 翻译练习¶

    把自然语言翻译成谓词逻辑是一个核心技能。设定：

    • S(x) = "x 是学生"
    • P(x) = "x 通过了考试"
    • T(x, y) = "x 教 y"
    • a = 张老师
    自然语言 谓词逻辑 解释
    所有学生都通过了考试 ∀x(S(x) → P(x)) 对任何 x，如果 x 是学生，x 就通过了
    有些学生没通过考试 ∃x(S(x) ∧ ¬P(x)) 存在某个 x，x 是学生且 x 没通过
    张老师教所有学生 ∀x(S(x) → T(a, x)) 对任何 x，如果 x 是学生，张老师教 x
    没有学生通过考试 ¬∃x(S(x) ∧ P(x)) 不存在一个既是学生又通过了考试的 x

    最后一句也可以等价地写成 ∀x(S(x) → ¬P(x))——"所有学生都没有通过"。

    4.5 用谓词逻辑重写三段论¶

    现在回到开头的论证：

    前提 1：∀x(H(x) → M(x))    — 所有人都会死
前提 2：H(a)                 — 苏格拉底是人
结论：  M(a)                 — 苏格拉底会死

    推导：从前提 1 知道"对所有 x"都成立，所以特别地对 a（苏格拉底）也成立：H(a) → M(a)。再加上前提 2 给出 H(a)，由肯定前件（modus ponens）得出 M(a)。

    这个论证在命题逻辑中无法表达其有效性，但在谓词逻辑中完全透明。

    可靠度：Level 1。谓词逻辑及其翻译方法是逻辑学教科书共识。

    5. 为什么弗雷格如此重要¶

    在弗雷格之前，西方逻辑学的主流工具是亚里士多德（Aristotle，公元前 384-322 年）建立的三段论（syllogism）——由两个前提和一个结论组成的固定论证格式。

    亚里士多德的三段论统治了逻辑学将近2000 年。它能处理"所有 A 是 B""有些 A 是 B"这类简单命题，但面对更复杂的关系和嵌套量词就力不从心。比如：

    "每个自然数后面都有一个更大的自然数"

    这句话涉及两个量词的嵌套（"对每个 x，存在一个 y，使得 y > x"），三段论根本无法表达。

    1879 年，德国数学家和逻辑学家弗雷格（Gottlob Frege, 1848-1925）发表了《概念文字》（Begriffsschrift，直译为"概念书写"）。这本薄薄的小书：

    1. 发明了量词（∀ 和 ∃ 的前身）——这是逻辑史上最重要的技术发明
    2. 建立了完整的谓词逻辑系统——包括推导规则和公理
    3. 展示了如何用逻辑分析数学推理——开启了逻辑主义（logicism）纲领：试图证明数学可以完全还原为逻辑

    这个贡献的重要性怎么估计都不为过。哲学家迈克尔·达米特（Michael Dummett）称弗雷格为"语言哲学之父"。没有弗雷格的谓词逻辑，分析哲学就不会以现在的形式存在。

    弗雷格的更多贡献（涵义与指称、逻辑主义纲领的兴衰）将在下一章详细展开（详见 03 章）。

    可靠度：Level 1。弗雷格在现代逻辑史上的地位是学术共识。

    可靠度：Level 2。将弗雷格视为"分析哲学之父"在学术界是主流观点，但也有学者认为这个头衔应该给罗素或穆尔（G.E. Moore）。
