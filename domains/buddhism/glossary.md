# 佛教术语表

| 缩写/术语 | 全称/原语 | 简述 |
|-----------|----------|------|
| abhidharma | 阿毗达磨 | 对佛教教义的系统化哲学分析 |
| ālaya-vijñāna | 阿赖耶识 | 唯识学派提出的第八识，储存一切经验"种子"的意识流 |
| anātman / anattā | 无我 | 否认恒常不变的自我存在 |
| anicca | 无常 | 一切有为法（由条件产生的事物）都是变化的 |
| arhat / arahant | 阿罗汉 | 已断除一切烦恼、获得解脱的人（早期佛教的理想人格） |
| Aśoka | 阿育王 | 古印度孔雀王朝皇帝，大力推广佛教 |
| ātman | 我 / 梵我 | 婆罗门教中永恒不变的灵魂/自我（佛教否认其存在） |
| avidyā / avijjā | 无明 | 不了解事物真实面目（缘起、无我），十二因缘的第一支 |
| bhava | 有 | 十二因缘第十支，由执取驱动的"成为"过程 |
| bīja | 种子 | 唯识学中行为留在阿赖耶识中的潜在影响力 |
| bodhi | 菩提 | 觉悟 |
| bodhicitta | 菩提心 | 为一切众生获得觉悟的发愿（大乘核心概念） |
| bodhisattva | 菩萨 | 发愿帮助一切众生成佛的修行者（大乘理想人格） |
| Brahman | 梵 | 婆罗门教中宇宙的终极实在 |
| catuṣkoṭi | 四句否定 | 龙树的论证方法：同时否定 P、¬P、P∧¬P、¬(P∨¬P) |
| cetanā | 思 / 意志 | 佛教认为业的本质就是意志（"我说业就是思"） |
| Chan / Zen | 禅 / 禅宗 | 中国佛教宗派，强调直接体验而非文字教导 |
| dependent origination | 缘起 / pratītyasamutpāda | 一切事物依条件聚合而生，无独立自存的实体 |
| dharma | 法 | 多义词：佛陀的教导 / 存在的基本元素 / 宇宙法则 |
| duḥkha / dukkha | 苦 | 不圆满性（unsatisfactoriness），佛教对人生处境的基本判断 |
| dvādaśanidāna | 十二因缘 | 缘起原理的具体展开，描述苦的产生链条 |
| Four Noble Truths | 四谛 / 四圣谛 | 苦、集、灭、道——佛教的逻辑框架 |
| Hīnayāna | 小乘 | 大乘对早期佛教的贬义称呼（学术上不使用） |
| karma / kamma | 业 | 佛教中指意志/动机及其后果 |
| kōan | 公案 | 禅宗教学工具，用看似荒谬的问题打破概念思维 |
| Kumārajīva | 鸠摩罗什 | 重要译经家（344-413），意译风格 |
| Madhyamaka | 中观 | 龙树创立的大乘哲学学派，核心主张"一切法空" |
| Mahāyāna | 大乘 | 佛教的主要分支之一，强调菩萨理想和普度众生 |
| manas | 末那识 | 唯识学中的第七识，恒常执阿赖耶识为"我" |
| mārga | 道 | 灭苦之路（四谛中的道谛 = 八正道） |
| MBSR | 正念减压 | 世俗化的正念治疗技术 |
| Nāgārjuna | 龙树 | 中观学派创始人（约 150-250），著《中论》 |
| nirodha | 灭 | 苦的止息（四谛中的灭谛 = 涅槃） |
| nirvāṇa / nibbāna | 涅槃 | 渴爱和无明的彻底止息 |
| Noble Eightfold Path | 八正道 | 灭苦的具体实践路径（正见到正定八项） |
| pañcaskandha | 五蕴 | 色受想行识——对人类经验的五种分析角度 |
| paramārtha-satya | 胜义谛 | 究竟层面的真理（一切法无自性） |
| pāramitā | 波罗蜜 | 圆满实践（六波罗蜜：布施、持戒、忍辱、精进、禅定、般若） |
| paratantra | 依他起性 | 唯识三性之一：事物依条件产生的实际存在方式 |
| parikalpita | 遍计所执性 | 唯识三性之一：错误执着的实在性 |
| pariniṣpanna | 圆成实性 | 唯识三性之一：对依他起性的正确认识 |
| Parfit | 帕菲特 | 英国哲学家，其人格同一性理论与无我论相似 |
| prajñāpāramitā | 般若波罗蜜多 | "智慧的圆满"——对空性的直接洞见 |
| Prāsaṅgika | 归谬论证派 | 中观学派分支，只用归谬法不立自宗 |
| pratītyasamutpāda | 缘起 | 同 dependent origination |
| rūpa | 色 | 物质性存在（五蕴之一） |
| samādhi | 定 / 三摩地 | 心理专注状态（三学之一） |
| saṃsāra | 轮回 | 众生在不同存在状态中的流转 |
| saṃvṛti-satya | 世俗谛 | 日常层面的真理（桌子存在、因果有效） |
| saṅgha | 僧团 / 僧伽 | 佛教修行者的团体 |
| saṅkhāra | 行 | 意志活动 / 心理活动（五蕴之一 / 十二因缘第二支） |
| saññā | 想 | 认知/分辨功能（五蕴之一） |
| Sarvāstivāda | 说一切有部 | 重要部派，主张"一切法三世实有" |
| Sautrāntika | 经量部 | 重要部派，只承认经的权威，引入"种子"说 |
| sīla | 戒 | 道德约束（三学之一） |
| skandha | 蕴 | 聚合——五蕴中的"蕴" |
| śramaṇa | 沙门 | 古印度反婆罗门的修行者运动 |
| śūnyatā | 空 / 空性 | 一切法无自性——不是虚无，是依缘而起 |
| sūtra / sutta | 经 | 佛陀教导的记录 |
| svabhāva | 自性 | 独立自足、恒常不变的本质（中观所否定的对象） |
| Svātantrika | 自立论证派 | 中观学派分支，认为可以用独立论证证明空性 |
| taṇhā | 渴爱 | 苦的直接原因——对感官对象的贪恋或厌恶 |
| tathāgatagarbha | 如来藏 | 一切众生内在的成佛潜力 |
| Theravāda | 上座部 | 佛教最古老的现存传统，主要在斯里兰卡和东南亚 |
| trilakṣaṇa | 三法印 | 无常、苦、无我——佛教核心哲学立场的标识 |
| trisvabhāva | 三性 | 唯识学的三种存在层面 |
| upāya | 方便 | 灵活的教学/度化方法 |
| Vasubandhu | 世亲 | 唯识学派重要哲学家 |
| vedanā | 受 | 感受——乐、苦、不苦不乐（五蕴之一） |
| vijñāna / viññāṇa | 识 | 意识/觉知功能（五蕴之一） |
| vijñapti-mātra | 唯识 | "仅是识的显现"——唯识学的核心命题 |
| vinaya | 律 | 僧团的规则和制度 |
| Xuanzang | 玄奘 | 唐代译经家（602-664），直译风格 |
| Yogācāra | 唯识学派 / 瑜伽行派 | 大乘佛教两大哲学学派之一，主张万法唯识 |
